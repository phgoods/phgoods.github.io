15. fungsi font ,font size,font color


Fungsi :

Font, berfungsi untuk mengganti jenis font pada dokumen. Umunya pada microsoft Office, font default yang digunakan adalah Calibri.Font Size, berfungsi untuk menggantu ukuran font sesuai keperluan pengguna. Ukuran font tersedia dalam rentang skala 8 - 72.Font Color, berfungsi untuk mengganti warna font sesuai keperluan pengguna. Umumnya akan disediakan standart colour, themed colour dan custom colour

Pembahasan

Font adalah salah satu menu pada Microsoft office yang berada pada ribbon Home yang terletak di nomor dua dari kiri tampilan Microsoft Office. Selain 3 perintah di atas, ada beberapa hal yang dapat dilakukan pada menu font yakni :

Increase or Decrease Font Size, digunakan untuk menaikkan atau menurunkan ukuran font pada satu skala diatas atau dibawahnya.Change Case, digunakan untuk mengubah model susunan font apakah Uppercase, Lowercase, sentece case, atau toggle caseClear all formatting, digunakan untuk menghilang semua efek hasil format terhadap font yang di selectText Effct dan typpgraphy, digunakan untuk membuat efek pada font seperti glow, outline, shadow dan sebagainya untuk memperindah dokumenText Highlight colour, digunakan untuk memberi highlight atau jika dalam konteks tertulis adalah stabilo. Warna juga dapat disesuaikanSubscript dan Superscript diguakan untuk memberi pangkat atau angka dibawah seperti pada kimiaBold, Italic, Underline, Stikethrough, secara berurutan digunakan untuk memberikan efek penebalan, memiiringkan tulisan, memberi garis bawah, dan mencoret tulisan yang diselect.

21. Perbedaan font size,grow font dan shrink font


Jawaban:

Misalkan pada program pengolah kata seperti Ms. Word 2007:

Font Size: digunakan untuk mengubah ukuran huruf dg mengisikan angka pada kotak isian yg disediakan (semakin besar angka, maka semakin besar ukuran huruf & sebaliknya).

Grow font: digunakan untuk mengubah ukuran huruf menjadi lebih besar secara otomatis.

Shrink font: digunakan untuk mengubah ukuran menjadi lebih kecil secara otomatis.

Penjelasan:

Pada program Ms. Word 2007, font size, grow font dan shrink font terdapat dalam tab ribbon Home di group Font. Perhatikan gambar berikut ini:

Font size: ditunjukkan tanda panah merah

Grow font: ditunjukkan tanda panah biru

Shrink font: ditunjukkan tanda panah hijau
