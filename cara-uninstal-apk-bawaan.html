17. Apa bahasa inggrisnya Saya sudah lama punya, tapi saya uninstal


TASK : Translate to English

Original : Saya sudah lama punya, tapi saya uninstall

Penjelasan :

1) Saya sudah lama punya (di revisi => saya sudah memilikinya untuk beberapa lama)

Saya = I

sudah memiliki = have had  => auxiliary verb + verb 3

(bentuk present perfect tense dimana have berfungsi sebagai auxiliary verb atau kata kerja bantu, sedangkan had sebagai bentuk lampau dari have)

untuk = for

beberapa lama = a while

2) tapi saya uninstall

tapi = but

saya = I

uninstall = mencopot (direvisi => uninstalled karena kalimat ini merupakan bentuk lampau)

Sehingga terjemahan dari kalimat saya sudah lama punya, tapi saya uninstall adalah :

I have had it for a while, but I uninstalled it

22. Apa yang dimaksud dengan uninstal​


Uninstall adalah menghapus atau menghilangkan perangkat lunak atau bagian-bagiannya dari komputer. Uninstall ini adalah kebalikan dari install.

Pembahasan:

Sistem komputer secara umum memiliki bagian berupa software (perangkat lunak) dan hardware (perangkat keras).

Hardware adalah perangkat keras komputer. Hardware meliputi perangkat seperti mouse dan keyboard, prosesor, penyimpan data seperti hard disk dan CD, serta perangkat output seperti printer, monitor dan speaker.

Software adalah perangkat lunak dalam komputer. Perangkat lunak menyediakan antarmuka dan fitur agar pengguna dapat dengan mudah menggunakan komputer. Contohnya adalah perangkat lunak pengolah citra seperti Adobe Photoshop atau perangkat lunak pengolah berkas Microsoft Word.

Untuk menggunakan perangkat lunak, maka biasanya perangkat lunak ini harus dipasangatau di-install terlebih dahulu. Untuk meng-install digunakan installer dari perangkat lunak tersebut.

Sebaliknya, setelah perangkat lunak tidak digunakan lagi, maka perangkat lunak ini dapat dihapus, dalam proses uninstall. Proses menghapus perangkat lunak ini juga membersihkan penyimpanan (hard disk), sehingga dapat digunakan untuk menyimpan berkas lain.

Pada smartphone, perangkat lunak ini disebut app (kependekan dari application atau aplikasi). Untuk meng-install dan meng-uninstall app, pada smartphone dengan dengan sistem operasi Android digunakan perantara yaitu aplikasi PlayStore, sementara untuk smartphone dengan dengan sistem operasi Apple digunakan aplikasi App Store.
