1. bagaimana cara menginstal game dari kaset ke pc


1. Masukan CD tersebut kedalam komputer / laptop.
2. Buka file tersebut, My Computer > CD 
3. Klik ganda pada setup.
4. Lakukan penginstalan
5. Menunggu.
6. Ketika selesai, restart komputer untuk menambah performa komputer.
7. Nikmati permainan.

Jika komputer membutuhkan dvd rom kalau leptop biasanya sudah ada atau menggunakan dvd eksternal , masukkan kasetnya , biasanya akan muncul autorun.exe / setup.exe jika tidak klik icon my computer lalu klik gambar kaset jika sudah terbuka tinggal klik next sampai selsai
