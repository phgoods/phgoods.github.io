2v + 3e = 1.200.000
3v + 4e = 1.700.000
disamakan dulu, menjadi
6v + 9e = 3.600.000
6v + 8e = 3.400.000
------------------------------ dikurangi
e = 200.000

mencari v dengan menggunakan persamaan 1
2v + 3(200.000)=1.200.000
2v = 1.200.000 - 600.000
v = 600.000/2
v = 300.000

jadi harga tiket Ekonomi = 200.000
harga tiket VIP = 300.000

2x + 3y = 1.200.000

2x + 3(200.000) = 1.200.000

21. Harga tiket suatu kapal wisata dengan kapasitas 70 kursi adalah Rp65.000 untuk orang dewasa dan Rp18.000 untuk anak-anak. Jika uang hasil penjualan tiket 1 perjalanan wisata adalah Rp1.824.000 berapakah banyaknya orang dewasa dan anak-anak yang mungkin dalam kapal tersebut?


Penjelasan dengan langkah-langkah:

Anak-anak = a

Orang Dewasa = b

a + b = 70

a = 70 - b

65.000 a + 18.000 b = 1.824.000

Dibagi 1.000

65a + 18b = 1.824

65 (70 - b) + 18b = 1.824

4.550 - 65b + 18b = 1.824

-47b = 1.824 - 4.550

b = -2.726/-47

b = 58

a = 70 - b

a = 70 - 58

a = 12

Banyak anak-anak = 12 orang

Banyak orang dewasa = 58 orang

29. Terdapat sebuah kapal antar pulau yang dapat mengangkut penumpang sebanyak 240 orang. Penumpang kelas utama dapat membawa bagasi 60kg dan kelas ekonomi 20kg. Kapal tersebut hanya dapat mengangkut bagasi 7.200kg. Harga jual tiket kelas utama RP100.000,- dan tiket ekonomi Rp50.000,-. Keuntungan maksimal yang dapat diperoleh oleh pengusaha kapal tersebut adalah


Fungsi Linier.

Misalkan:

kelas utama = x

kelas ekonomi = y

Fungsi Objective:

100.000x + 50.000y

Fungsi Pembatas:

x + y ≤ 240

60x + 20y ≤ 7.200 → 3x + y ≤ 360

x ≥ 0

y ≥ 0

Penyelesaian:

1. Menentukan titik potong sumbu x dan sumbu y dan titik potong kedua fungsi

a. Fungsi x + y ≤ 240

→ titik potong sumbu x ( y = 0 )

x + 0 = 240

x = 240

titik koordinat titik potong sumbu x (240, 0)

→titik potong sumbu y ( x = 0 )

0 + y = 240

y = 240

titik koordinat titik potong sumbu x (0, 240)

b. fungsi 3x + y ≤ 360

→ titik potong sumbu x ( y = 0 )

3x + 0 = 360

x = 120

titik koordinat titik potong sumbu x (120, 0)

→titik potong sumbu y ( x = 0 )

0 + y = 360

y = 360

titik koordinat titik potong sumbu x (0, 360)

c. Kedua fungsi

x + y - 240 = 3x + y - 360

360 - 240 = 3x - x + y - y

120 = 2x

x = 60

y = 240 - 60 = 180

titik potong kedua fungs ( 60, 180)

2. Gambarkan kedua fungsi di atas, kemudian tentukan titik - titik pojoknya.

(lihat gambar pada tautan)

dari gambar dapat kita tentukan 3 titik pojok yang memenuhi batas - batas dari kedua buah fungsi tersebut yaitu:

(0,240) (60,180) dan (120,0)

3. Menentukan nilai maximum dan minimum fungsi objective

nilai maximum dan minimum dapat ditentukan dengan menguji nilai yang dihasilkan titik titik pojok tersebut terhadap fungsi objective.

a. titik (0,240)

100.000x + 50.000y

= 100.000(0) + 50.000 (240)

= 12.000.000

b. titik (60,180)

= 100.000 (60) + 50.000 (180)

= 6.000.000 + 9.000.000

= 15.000.000

c. titik (120,0)

= 100.000 (120) + 50.000 (0)

= 12.000.000

sehingga hasil maximum adalah 15.000.000 dengan komposisi 60 tiket kelas utama dan 180 tiket kelas ekonomi.
