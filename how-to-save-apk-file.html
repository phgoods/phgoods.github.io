Pembahasan

Perintah-perintah keyboard ada banyak. Semua perintah ini berperan dalam membantu kita. Untuk itu, ada beberapa perintah dari keyboard.

Ctrl + A, berguna untuk select all atau memblok semua text

Ctrl + B, berguna untuk menebalkan word atau huruf (bold)

Ctrl + C, berguna untuk mengcopy text atau gambar

Ctrl + D, memilih jenis huruf

Ctrl + E, rata tengah (center)

Ctrl + F, untuk mencari text

Ctrl + G, untuk perintah go to

Ctrl + H, untuk mencari dan memblok text (find and replace), pada chrome bisa digunakan untuk fungsi history

Ctrl + I, untuk perintah miring (italic)

Ctrl + J, untuk perintah rata atau seimbang antara kiri dan kanan (Justify) atau dalam chrome bisa digunakan file yang di download

Ctrl + K, untuk memasukkan alamat tautan (hyperlink) atau dalam chrome untuk penelusuran goggle

Ctrl + L, untuk membuat rata kiri (Left) atau dalam chrome memblok alamat tautan

Ctrl + M, menggeser paragraf ke kanan

Ctrl + N, untuk membuat dokumen baru (New)

Ctrl + O, untuk membuka berkas (Open) atau pada chrome untuk mencari file dan menyimpan sesuatu

Ctrl + P, untuk mencetak suatu dokumen

Ctrl + Q, untuk menghapus setting

Ctrl + R, untuk rata kanan

Ctrl + S, untuk menyimpan berkas (Save)

Ctrl + T, untuk menggeser tabulasi ke kanan atau pada chrome untuk membuka tab baru

Ctrl + U, untuk garis di bawah tulisan (Underline)

Ctrl + V, untuk menempel apa yang kita copy (Paste)

Ctrl + W, untuk keluar dari suatu dokumen

Ctrl + X, untuk memotong atau mendelete text

Ctrl + Y, untuk melakukan redo atau mengembalikan perintah

Ctrl + Z, untuk membatalkan perintah (Undo)
