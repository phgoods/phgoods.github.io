1. pengertian internal backup dan network backup


Internal backup biasanya mencakup backup data yg di lakukan secara manual, biasanya menggunakan flashdisk/harddisk external sebagai media backup-annya.

Network backup biasanya melalui jaringan internet, misalnya Google Drive, i-Cloud, Mi Cloud, dll..


2. Apakah perbedaan antara full backup dengan incremental backup


Incremental backup adalah menyalin semua data yang berubah sejak terakhir kali melakukan full backup

Full backup adalah menyalin semua data termasuk folder ke media lain.
