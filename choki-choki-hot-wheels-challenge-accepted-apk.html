17. umur ani adalah ½ umur beni dan ⅓ umur choki,jumlah umur mereka 84 tahun tentukan umur mereka tolong beserta caranya trima kasih


misal: ani = A ; beni = B dan choki = C

A = 1/2 B
A = 1/3 C ===> 1/2 B = 1/3 C
                                C = 1/2 B / 1/3
                                C = 3/2 B

A + B + C = 84 ===> 1/2 B + B + 3/2 B = 84
                                                          3B = 84
                                                            B = 28

A = 1/2 B ===> A = 1/2 x 28
                         A = 14

C = 3/2 B ===> C = 3/2 x 28
                         C = 42

Jadi umur Ani = 14 thn, umur Beni = 28 thn dan umur Choki = 42 thn

21. Kecepatan mobil choki 90km/jam. Jarak sukabumi-tasik 60km. Jika Choki berangkat dari sukabumi pukul 10.00. Pukul berapa tiba di tasik?


PEMBAHASAN

Diketahui:

K = 90 km/jam

J = 60 km

berangkat = 10.00

Ditanya:

tiba = ?

Jawaban:

Menentukan lama perjalanan

[tex]w = \frac{j}{k} [/tex]

[tex]w = \frac{60 \: km}{90 \:km/jam } [/tex]

[tex]w = \frac{60}{90} \: jam[/tex]

[tex]w = \frac{(60 \div 30)}{(90 \div 30)} \: jam[/tex]

[tex]w = \frac{2}{3} \: jam[/tex]

[tex]w = \frac{2}{3} \times 60 \: menit[/tex]

[tex]w = 2 \times 20 \: menit[/tex]

[tex]w = 40 \: menit[/tex]

Menentukan waktu tiba

tiba = berangkat + lama

tiba = 10.00 + 40 menit

tiba = 10.00 + 00.40

tiba = 10.40

Kesimpulan:

Jadi, Choki akan tiba di Tasik pada pukul 10.40.

Diketahui:

Kecepatan (K) = 90 km/jamJarak (J) = 60 kmBerangkat pada pukul 10.00

Ditanyakan:

Tiba pada pukul?

Penyelesaian:

Menentukan waktu perjalanan

W = J/K

= 60/90 (masing² bagi 30)

= 2/3 jam

Karena 2/3 jam kurang dari 1 jam, kita dapat mengubahnya menjadi satuan menit dengan mengalikannya dengan 60 (1 jam = 60 menit).

W = ⅔ × 60

= 2 × 20

= 40 menit

Menghitung pukul tibanya

Cara menghitungnya adalah dengan menjumlahkan pukul berangkat dengan waktu perjalanan. Sehingga:

Pukul tiba = 10.00 + 00.40

= 10.40

Kesimpulan:

Jadi, Choky akan tiba di Tasik pada pukul 10.40
