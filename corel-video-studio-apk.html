27. sebutkan 5 lintasan yang ada di corel video studio


Lintasan yang ada di Corel Video Studio adalah sebagai berikut

File Tab

- Ctrl + N = New Project

- Ctrl + M = New HTML5 Project

- Ctrl + O = Open Existing Project

- Ctrl + S = Save Project

- Alt + F4 = Close

Edit Tab

- Ctrl + C = Copy Clip

- Ctrl + X = Cut Clip

- Ctrl + V = Paste Clip

- Ctrl + Z = Undo

- Ctrl + Y = Redo

- Ctrl + A = Select All Clips

- + = Zoom in

- - = Zoom Out

Setting Tab

- F6 = Preferences

- Alt + Enter = Project Properties

- F1 = Help

Runtime Shortcut

- Ctrl+P = Play/Pause Clip

- Spacebar = Play/Pause Clip

- Shift Continous = Trimmed Play Clip

- Shift+Play Click = Play selected Clip

- Ctrl+R = Repeat Play

- Ctrl+L = Volume

- Ctrl+I = Split videos

- Shift+Click = Select Multi Clips

- Panah Kiri = Select Previous Clip

- Panah Kanan = Select Next Clip

- Ctrl + Panah Kanan = Scroll Forward

- Ctrl + Panah Kiri = Backword

- Ctrl + Panah Atas = Atas

- Ctrl + Panah Bawah = Bawah

- F6 = Go Forward

- F5 = Go backward

- Esc = Cancel

- Del = Delete

- F3 = Mark In

- F4 = Mark Out

Navigation / Direction Shortcut

-  Home = Kembali ke Awal Segmen

- End = Ke Akhir Segmen

- Panah Atas = Ke Step Sebelumnya

- Panah Bawah = Ke Step Selanjutnya

- Alt+E = Edit

- Alt+C = Capture

- Alt+S = Share

- Alt+F = Effect(Gallery)

- Alt+O = Media(Gallery)

- Alt+T = Title(Gallery)

- B = Frame Sebelumnya

- F = Frame Selanjutnya

- Ctrl+H = Home

- Ctrl+E = End

- Tab =  Select Scrubber/Trim handle
