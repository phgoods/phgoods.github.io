13. cara install file tar.gz​


Jawaban:

Bagaimana cara menginstallnya?

PERINGATAN! bagi yang masih pemula di linux tidak disarankan memakai cara ini karena agak susah ,gunakan package manager pada distro masing2 spt: synaptic dan USC pada Ubuntu. selengkapnya lihat disini

Langkah-langkahnya sbb:

extrak file .tar.gz tsb dng perintah "tar -xzfvf nama_file.tar.gz"

masuk ke folder hasil extraksi dengan perintah "cd nama_folder"

jalankan perintah "./configure"

jalankan perintah "make"

jalankan perintah "make install"

Perintah ./configure berfungsi untuk memeriksa ketersediaan library pendukung dan untuk mempersiapkan segala sesuatunya agar source kode siap dikompilasi. Jika library yang dibutuhkan untuk meng-compile tidak tersedia biasanya akan terjadi error pada perintah ./configure, solusinya install library yang dibutuhkan tsb, jika masih bingung silakan post pertanyaan anda disini

sedangkan perintah make berfungsi untuk meng-compile source code menjadi file binary yang siap untuk dijalankan, dan make install merupakan perintah untuk melakukan instalasi program.

IDE adalah lingkungan pengembangan terintegrasi untuk pengembangan program saat kita memrogram atau mengode (coding) dengan dilengkapi alat-alat untuk debug dan eksekusi.
