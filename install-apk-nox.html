8. jelaskan perbedaan dari options penginstalan install dan graphical install !


Jelaskan perbedaan dari options penginstallan install dan graphical install, pada saat kita akan melakukan proses installasi sistem operasi, kita biasanya membuat isntaller terlebih dahulu menggunakan media dvd atau flashdisk. Setelah melakukan pemilihan ke dalam daftar booting pada BIOS, maka selanjutnya akan muncul proses penginstallan. Pada proses penginstallan ini biasanya kita akan diberikan pilihan yaitu mode graphic atau install menggunakan teks.

Biasanya untuk proses penginstallan menggunakan teks ada pada sistem operasi Linux. Sedangkan pada windwos sudah menggunakan graphical install semua. Perbedaannya apabila menggunakan graphical akan sangat mudah karena bisa menggunakan mouse dan keyboard, seperti menggunakan komputer biasa dan menginstall aplikasi. Sedangkan menggunakan mode teks, hanya bisa menggunakan keyboard saja, mulai dari memilih partisi membuat partisi dan hingga langkah terakhir.

10. kadar gas pencemar NOx di suatu daerah perkotaan dengan jumlah kendaraan yang tinggi dapat mencapai 0,005% kadar gad NOx tersebut jika dinyatakan dalam suatu ppm adalah


[NOx] : 0,005% (v/v)
[NOx] : ....... ppm ?

Jawab

ppm : part per-million

Satuan ppm untuk sampel gas adalah μL/L

0,005% (v/v) artinya 0,005 mL NOx dalam 100 mL larutan.

ppm : mL / L
ppm : 0,005 mL x 10^3 μL/mL / 100 mL x 10^-3 L/mL
: 5 μL / 0,1 L
: 50 μL/L
: 50 ppm

Rumus cepat untuk menentukan ppm jika diketahui %

ppm : % . 10^4
ppm : 0,005% . 10^4
: 50 ppm
