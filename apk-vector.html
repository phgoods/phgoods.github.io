6. Diketahui vector a = 4i – 2j + 2k dan vector b = 2i – 6j + 4k. Proyeksi vector orthogonal vector a pada vector b adalah .....


vektor a = 4i – 2j + 2k

vektor b = 2i – 6j + 4k

a . b = (4).(2) + (-2).(-6) + (2).(4) = 8 + 12 + 8 = 28

|b|² = 2² + [-6]² + 4² = 4 + 36 + 16 = 56

Proyeksi vector orthogonal vector a pada vector b :

= ( [a . b] / |b|² ) . vektor b

= (28 / 56) . vektor b

= ½ . vektor b

= ½ . (2i – 6j + 4k)

= i - 3j + 2k

• komponen vektor


V = 12 N, α = 30°

V₁ = __? V₂ = __?



komponen vektor V


V₁ = V cos 30°

V₁ = 12 • ½√3

V₁ = 6√3 N


V₂ = V sin 30°

V₂ = 12 • ½

V₂ = 6 N
