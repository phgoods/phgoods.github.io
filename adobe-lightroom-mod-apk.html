8. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.

16. bagaimana cara mendownload preset lightroom secara gratis?​


Jawaban:

1.Download terlebih dahulu file preset yang sudah disediakan. Selanjutnya ekstrak file preset dan taruh di folder yang mudah di temukan.

Download File Preset

2. Buka Lightroom mobile kemudian tap menu New item lalu tap lagi menu Album untuk membuat album baru.

3. Beri nama pada album misalnya saja “preset” berikutnya tap icon picture untuk mulai menambahkan preset.

4. Tap menu Device lalu pilih folder preset yang sudah kita ekstrak sebelumnya kemudian tap tombol Add.

5. Pilih salah satu preset yang ada di album, lalu tap 3 titik di pojok kanan atas untuk memunculkan menu yang lain.

6. Langsung tap menu Copy Settings lalu tap kembali dan masukkan gambar atau foto yang ingin kamu edit. Tap lagi pada 3 titik yang ada di pojok atas.

7. Tap Paste Setting untuk memasukkan preset pada gambar.

Demikian tutorial cara mudah memasukkan preset Lightroom di Android. Semoga bermanfaat.
