29. Bagaimana cara mempelajari grammar ? Untuk katanya bebas 1. Pro / hv / v / art / n / prep / pro 2. Pro / prep / art / n / v / adj 3. V / art / n / advb 4. N / conj / n / hv / v / prep / n 5. Art / n / v / art / n


ENGLISH GRAMMAR

 

Jawaban  

 

Pendahuluan  

Dalam sebuah kalimat terdapat beberapa elemen. Beberpa dari elemen tersebut adalah part of speech. mereka antara lain:

N = noun (kata benda) seperti: man, woman, chair, book, dll

V = verb (kata kerja) seperti: walk, run, listen, speak, dll

 Prep = Preposition (kata depan) seperti: on, in, above, between, dll

Art = Article (artikel) seperti, a, an, the

pro = pronoun (kata ganti) seperti: he, she, him, us, your, mine

adj = Adjective (kata sifat) seperti: hard, soft, handsome, dll

adv = Adverb (kata keterangan) seperti: at home, on monday, carefully, dll

hv = helping verb (kata kerja bantu) seperti: can, may, will, did, do, has, have, is, are dll

conj = conjunction (kata hubung) seperti: and, or, because, dll

Pembahasan  

Buatlah kalimat dengan pola sebagai berikut:

1. Pro / hv / v / art / n / prep / pro


She can solve the problem of mine.

2. Pro / prep / art / n / v / adj


Mine on the shelf got burned.

3. V / art / n / advb


Do the dishes now!

4. N / conj / n / hv / v / prep / n


Plate and glass cab be placed in the shelf.

5. Art / n / v / art / n

 The man hits a car.
