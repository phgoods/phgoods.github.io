9. fungsi dialog box launcher pada ms power point 2013 ​


Jawaban:

Fungsinya untuk membuka kotak dialog (dialog box) secara lebih cepat.

Penjelasan:

Ms power point 2013 ​memiliki banyak tab ribbon, salah satunya ribbon Home. Dalam ribbon ini terdapat dialog box launcher pada group Font & Paragraph.

Bila tombol itu diklik maka akan tampil kotak dialog (dialog box) Font & kotak dialog (dialog box) Paragraph.

Perhatikan gambar berikut ini:

tanda panah nomer 1 menunjukkan dialog box launcher di group Font

tanda panah nomer 2 menunjukkan dialog box launcher di group Paragraph
