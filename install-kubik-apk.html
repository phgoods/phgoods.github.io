23. mengapa kita harus menginstall modem terlebih dahulu


FUNGSI INSTALASI DRIVER PADA MODEM Jawaban

Kita harus menginstal modem terlebih dahulu agar modem tersebut dikenali oleh komputer kita dan komputer dapat menjalankan modem tersebut tanpa terkendala.

Pembahasan

Setiap perangkat keras di dalam komputer pasti memiliki sebuah “manual” yang sering disebut dengan software driver. Apa itu driver? Driver adalah sebuah komponen penunjang dalam komputer yang bersifat khusus. Driver dapat dikatakan sebagai jembatan komunikasi antara hardware dengan sistem operasi. Dengan kata lain, agar sistem operasi dapat menjalankan perangkat keras dengan baik, maka kita perlu melakukan instalasi driver terlebih dahulu. Tanpa driver, perangkat keras tidak akan dapat dijalankan oleh sistem operasi.

Macam – macam software driver

Ada berbagai macam software untuk instalasi driver, diantaranya:

- DriverPack Solution

- Driver Booster

- DriverMax

- Driver Easy

Pilihlah software yang sesuai dengan kebutuhan.

Macam – macam driver

- Driver Audio: berhubungan dengan komponen audio atau suara

- Driver Network: berhubungan dengan komponen jaringan (LAN, Wireless)

- Driver Keyboard: berhubungan dengan komponen keyboard

- Driver Graphics: berhubungan dengan komponen visual (Biasa disebut driver VGA)

- Driver BIOS: berhubungan dengan motherboard

- Driver Storage: berhubungan dengan media penyimpanan seperti hard disk, flash disk, dll.

28. jelaskan perbedaan dari options penginstalan install dan graphical install !


Jelaskan perbedaan dari options penginstallan install dan graphical install, pada saat kita akan melakukan proses installasi sistem operasi, kita biasanya membuat isntaller terlebih dahulu menggunakan media dvd atau flashdisk. Setelah melakukan pemilihan ke dalam daftar booting pada BIOS, maka selanjutnya akan muncul proses penginstallan. Pada proses penginstallan ini biasanya kita akan diberikan pilihan yaitu mode graphic atau install menggunakan teks.

Biasanya untuk proses penginstallan menggunakan teks ada pada sistem operasi Linux. Sedangkan pada windwos sudah menggunakan graphical install semua. Perbedaannya apabila menggunakan graphical akan sangat mudah karena bisa menggunakan mouse dan keyboard, seperti menggunakan komputer biasa dan menginstall aplikasi. Sedangkan menggunakan mode teks, hanya bisa menggunakan keyboard saja, mulai dari memilih partisi membuat partisi dan hingga langkah terakhir.
