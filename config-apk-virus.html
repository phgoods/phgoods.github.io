Pengertian config adalah kepanjangan dari configuration yang memiliki artian sebagai pengaturan. Kemudian, server sendiri juga dianggap sebagai sebuah software yang melakukan pelaksanaan terhadap tugas tertentu. Akan tetapi hardware yang melakukan pendukungan terhadap software juga disebut sebagai server dikarenakan software server akan terhubung dengan ribuan client.

30. untuk melihat indikasi pada konfigurasi IP yang terpasang oada komputer dengan sistem operasi windows dengaa perintah?a. if configb. ip configc. is konfigd. IS config


Perintah yang digunakan untuk melihat indikasi pada konfigurasi IP yang terpasang pada komputer dengan sistem operasi Windows adalah IP Config.

Pembahasan

IP atau yang juga dikenal dengan sebutan Internet Protocol adalah serangkaian angka dan titik yang disematkan kepada setiap unit komputer dan perangkat jaringan. Konfigurasi angka dan titik ini berfungsi layaknya alamat pada jaringan komputer. Dengan kata lain, IP sebuah komputer atau perangkat jaringan bersifat unik atau terbatas untuk perangkat itu sendiri. Dengan demikian, ketika terhubung ke sebuah jaringan, sebuah komputer atau perangkat jaringan dapat dikenali secara khusus berkat IP yang tersemat kepadanya.

Pada perkembangannya, alamat atau identitas unik sebuah komputer atau perangkat jaringan tidak hanya terbatas pada IP melainkan juga pada MAC address yang berfungsi kurang lebih sama, namun MAC address turut mengombinasikan huruf selain angka dan titik.
