1. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.

11. apa yang di maksud extension*.gif


.gif (Graphics Interchange Format) adalah format gambar yang sudah cukup lama digunakan dan salah satu yang umum dipakai di internet.

Semoga Membantu!

Ekstensi file GIF (Graphics Interchange Format) adalah format gambar bitmap yang dirilis oleh CompuServe pada tahun 1987.

Format gambar ini mendukung sampai dengan 8 bit per piksel, memungkinkan sebuah gambar untuk merefrensikan palet warnanya sendiri sampai dengan 256 warna berbeda.

Fitur GIF yang paling dikenal adalah ekstensi GIF memungkinkan animasi atau gambar bergerak.

25. Apa kelebihan dan kekurangan gif


GIF: Graphics Interchange Format

PROS
- 256 jumlah warna
- menggunakan kompresi lossless
- Cocok untuk logo, bidang warna datar dengan daerah yang terdefinisi dengan baik
- Cocok untuk ikon dengan beberapa warna
- mendukung transparansi
- Cocok untuk animasi kecil

CONS
- format tertua untuk web - 1989
- dalam banyak kasus memiliki ukuran file yang lebih besar dari PNG3. GIF (Graphics Interchange Format)
Type file GIF memungkinkan penambahan warna transparan dan dapat digunakan untuk membuat animasi sederhana, tetapi sekarang ini standar GIF hanya maksimal 256 warna saja.
Kelebihan :

Mendukung animasi gambar
Bersifat tidak pecah-pecah
Menggunakan kompresi tidak menghilangkan data


Kekurangan :

Tidak cocok digunakan dalam fotografi modern
Penyimpanan data gambar terbatas
Gambar GIF berukuran lebih kecil dari pada JPG
