7. diketahui seseorang sedang mengupload file 48mb dengan kecepatan transfers 12kbps berapakah waktu yang dibutuhkan untuk mengupload.....menit?


Versi Kbps(kilo byte per second)

48MB = 48000KB
48000KB / 12 = 4000 Detik
4000 Detik / 60 = 66.666 Menit

Versi Kbps(kilo bits per second)
48MB = 48000KB
12 Kilobit / 8 = 1.5Kilobyte
48000KB  / 1.5 KB = 32000 Detik
32000 Detik / 60 = 533 Menit

kilo bits atau kilo byte ukuran digunakan tergantung dari program apa yang digunakan untuk mengukur kecepatan internet kalo agan menggunkanan modem dan tulisannya bisa 1mbps itu menggunakan satuan Bits.
