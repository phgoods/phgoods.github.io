16. perbedaan dari beberapa istilah JavaScript & WMLScript ?


Jawaban: JavaScript adalah bahasa pemrograman peramban, atau setidaknya, hanya itu yang terjadi hingga saat ini – jangkauannya kini telah menyebar di luar batas jendela peramban ke penggunaan lain seperti kode sisi server.

Dalam tujuan utamanya sebagai bahasa pemrograman sisi klien yang ditulis ke dalam setiap browser internet, ia bertanggung jawab atas elemen interaktif dan dinamis pada halaman web, seperti mendengarkan dan bereaksi terhadap input pengguna.

WML (Wireless Markup Language) yang mirip dengan HTML. WML akan mengubah informasi berupa teks dari halaman situs dan menampilkannya kelayar ponsel. WML juga memungkinkan pengguna menjelajahi suatu aplikasi WAP serta mendukung link seperti yang lazim terdapat di halaman web. WML juga merupakan bagian XML, maka WML didefinisikan berdasarkan aturan XML. Jadi, WML adalah suatu aplikasi XML.

Jika HTML mempunyai Java Script untuk membuat halaman-halaman di dalamnya agar interaktif, maka WML juga mempunyai WML Script yang dirancang untuk menambahkan fungsionalitas pada content statis dari WML, misalnya WML Script mampu melakukan validasi terhadap masukan (input) yang diberikan user dan berinteraksi dengan user.

Namun, berbeda dengan Java Script yang bisa dimasukkan ke dalam HTML, WML Script tidak bisa dijadikan satu dengan WML dan harus berdiri sendiri sebagai satu file skrip khusus yang berekstensi *.wmls. Perbedaan lainnya adalah gambar yang tampil di layar ponsel haruslah gambar yang telah dikonversi ke dalam format Wireless BitMap (wbmp) yang mempunyai kedalaman warna satu digit (dua ‘warna’ yaitu hitam dan putih).

Inti dari WML Script adalah pernyataan if (pernyataan bersyarat) dan for (perulangan) sebagai elemen penting dari suatu skrip. Dan WML Script juga merupakan skrip yang berorientasi pada fungsi, sehingga dalam suatu file WML Script dapat memuat banyak fungsi sekaligus.

19. Bagaimana membuat function pada javascript?


Jawaban:

Pakai keyword function lalu tuliskan nama function yang mau dibuat. Seperti berikut:

function namaFunc(argumentJikaAda) {

berikan statement disini

}

Bisa juga menggunakan arrow function (ES6) dengan cara berikut:

const namaFunc = (argumentJikaAda) => {

berikan statement disini

}

const merupakan arti dari konstanta yang tidak bisa diubah, jadi kalau ingin membuat function dan gabisa diubah orang asing bisa pakai const, jadi cuma bisa diubah di declarationnya atau pas dideklarasi. Tapi kalau mau biar bisa diubah bisa pakai var atau let. Kalau pakai arrow function ini juga bisa dikasih async di dekat argument. Kalau mau cari tau tentang async bisa check aja di Internet mengenai asynchronous.

Penjelasan:

Aturan penggunaan nama function sama dengan membuat variable. Yaitu:

• Tidak boleh ada spasi

• Tidak boleh memakai symbol yang berhubungan dengan operasi atau disebut operator seperti contohnya +,-,*,/,%

• Bisa memakai underscore

• Tidak boleh diawali nomor seperti 123iniVariable tetapi bisa jika nomornya tidak di awal nama variable seperti variable123

• Biasanya nama variable atau function menggunakan camelCase dibanding underscore jika ada spasi contohnya cari_angka_terbesar diganti menjadi cariAngkaTerbesar.

24. jelaskan apa saja data yang dapat diolah oleh javascript


Tipe data pada javascript meliputi: String, integer, float, array, object dan Booleans. Tipe data string adalah data yang memuat karakter, misalnya: “Semarang”, Teks yang ada di dalam tanda petik baik petik tunggal maupun petik ganda

Tipe data integer dan float merupakan tipe data numeric, tipe data object digunakan kurung kurawal (,.. -)

Setiap property pada tipe data object dipisahkan dengan tanda koma (,)

Tipe data Booleans terdiri dari 2 nilai yaitu: True and False

CONTOH:

Var x1=34;         //tipe data integer
