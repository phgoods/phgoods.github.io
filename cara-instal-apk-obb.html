12. cara menginstal mikrotik ??


1. masukkan cd mikrotik ke dalam cd/dvd room.
2. Setting bios komputer anda, pada booting awal (first boot)nya adalah cd/dvd room anda
.3. setelah di setting maka komputer anda akan booting pertama kale ke cd/dvd room anda.
4. Pencet tombol “A” untuk memilih semua konfigurasi. klik tombol “M” untuk memilih konfigurasi minimal. klik tombol “SPACE” untuk memilih konfigurasi secara manual.
5. jika sudah yakin maka klik yombol “I” untuk memulai menginstal.
6. Jika muncul seperti gambar diatas maka kita disuruh memilih memakai konfigurasi yang lama (jika sudah pernah menginstal mikrotik) atautidak.Klik tombol “Y” untuk menyetujui, atau klik tombol “N” untuk tidak menyetujui.
7. Setelah itu akan muncul peringatan yang maksudnya adalah memberitahu kita bahwa semua data akan hilang dan menanyakan apakah kita yakin atau tidak. Klik tombol “Y” untuk menyetujui, atau klik tombol “N” untuk tidak menyetujui.
8. jika muncul seperti gambar dibawah ini maka proses instalasi telah selesai. keluarkan kaset mikrotik anda, setelah itu tekan enter untuk merestart komputer anda secara otomatis.
9. Setelah komputer anda restart maka anda akan diminta untuk check disk atau tidak. Klik tombol “Y” untuk menyetujui, atau klik tombol “N” untuk tidak menyetujui.
10. setelah itu akan muncul tampilan login. “admin” (tanpa tanda petik) pada Mikrotik Login. Dan pada password enter aja. karena password defaultnya tidak ada password,,,,
11. Jika berhasil maka akan muncul gambar tampilan awal mikrotik (lihat gambar dibawah ini). Menandakan anda telah berhasil menginstal mikrotik.

25. Cara menginstal debian


1.       Hidupkan komputer.
2.       Setelah itu masukkan DVD ke dalam DVDroom.
3.       Masuk ke bios dan ubah firstbootnya menjadi DVD.
4.       Lalu kita mulai menginstall debian berbasis text.
5.       Saat muncul tampilan Debian 6, kita pilih install untuk memulai instalasi.
6.       Pada pilihan language, kita pilih saja English.
7.       Lalu kita pilih area posisi kita, pilih saja Other.
8.       Saat muncul tulisan Continent or Region pilih saja  Asia.
9.       Kemudian kita pilih bangsa kita yaitu indonesai.
10.   Saat muncul tulisan country to base default locale settings on pilih saja United States.
11.   Lalu kita pilih bahasa apa untuk keyboard kita, pilih sajaAmerican English.
12.   Tunggu sampai proses berhenti.
13.   Setelah itu akan muncul perintah untuk configurasi network, pilih saja Configure Network Manually.
14.   Kemudian isi IP Address untuk komputer kita, isikan IP sebagai berikut 192.168.1.1 karena kita menggunakan IP Address kelas C.
15.   Pada pengisian netmask isikan sebagai berikut255.255.255.0 karena kita menggunakan IP Address kelas C.
16.   Setelah pengisian netmask maka kita akan diperintah untuk mengisikan gateway untuk komputer kita, isikan gateway sebagai berikut 192.168.1.1 .
17.   Lalu kita isi name server address INGAT!!! menggunakanIP Address bukan menggunakan nama, karena disitu ada keterangan NOT HOST NAMES. Isikan dengan IP Adress sebagai berikut 192.168.1.1 .
18.   Kemudian Isikan Host Name dengan nama yang akan gunakan sebagai nama Super User, contoh saya akan menggunakan nama wahyu.
19.   Setelah itu kita isi Domain Name dengan nama yang Anda inginkan, saya menggunakan nama wahyu . com .
20.   Anda akan diperintahkan untuk memberi password pada Nama Super User Anda. Maka isilah sesuka Anda.
21.   Isikan ulang password Anda.
22.   Lalu Anda masukkan Nama User baru sesuka Anda, sebagai contoh saya menggunakan nama tekaje.
23.   Masukkan lagi nama User baru Anda.
24.   Setelah itu masukkan password untuk nama User baru Anda.
25.   Masukkan lagi password Anda.
26.   Setelah itu pilih kota yang Anda tempati, pilih sajaJakarta.
27.   Tunggu sampai proses yang sedang berjalan selesai.
28.   Lalu pilih Manual supaya kita kita mengkonfigurasi secara manual.
29.   Setelah itu pilih tulisan yang ada keterangan harddisknya.
30.   Kemudian pilih yes untuk menghapus seluruh partisi pada harddisk.
31.   Untuk membuat partisi pada harddisk pilih tulisan yang ada tulisannya FREE SPACE, lalu pilih Create a New Partition, lalu atur ukuran untuk root menjadi 2 GB, kemudian pilih yang Primary.
32.   Lalu kita pilih yang Beginning, karena kita membuat untuk awal bukan akhir.
33.   Kemudian pilih Done setting up the partition.
34.   Karena debian wajib menggunakan 2 partisi yaitu root dan swap, maka kita buat partisi lagi untuk swap.
35.   Pilih tulisan yang ada tulisannya FREE SPACE, lalu pilihcreate a new partition, kemudan Atur ukuran swap untuk debian kita menjadi 4 GB, setelah itu pilih yang Primary dan pilih Beginning.
36.   Kemudian kita ubah partisi untuk swap seperti Use Us menjadi swap area.
37.   Lalu pilih Done setting up the partition .
38.   Setelah selesai kita pilih Finish Partitions untuk mengakhiri pembuatan partisi.
39.   Klik yes pada saat muncul tulisan Write the Changes to disks.
40.   Tunggu sampai proses instalasi base system selesai.
41.   Pilih no pada scan another cd or DVD.
42.   Pilih no pada Use a Network Miror.
43.   Pilih no pada Participate in the Package usage Survey.
44.   Kasih bintang  pada Standart System Utilities saja. Jika menambah atau mengurangi tanda bintang Anda tinggal tekan space.
45.   Pada saat muncul tulisan GRUB Boot Loader pilih sajayes.
46.   Instalasi telah selesai pilih saja Continue.
47.   Setelah itu ambil DVD dan nanti akan restart sendiri.

29. bagaimana cara menginstal leptop


1. Nyalakan laptop, kemudian masuk ke menu bios dengan cara menekan tombol 'del' saat laptop mulai menyala.

2. Setelah masuk ke menu bios, pilihlah menu boot. Di sini kita harus memilih boot device dengan urutan pertama adalah CD/DVD ROM, dan save setting bios. Jadi pada saat laptop dinyalakan, otomatis akan membaca CD/DVD ROM yang sudah dimasukkan CD Windows, dan kemudian akan berlanjut cara instal laptop tahap selanjutnya ini.

3. Masukkan CD Windows ke dalam CD/DVD Rom, kemudian restart laptop.

4. Ketika laptop baru menyala, tunggu beberapa saat dan akan muncul tulisan "press any key bla bla bla..." tekan saja sembarang tombol di keyboard, (enter misalnya).

5. Proses loading... kemudian akan muncul tiga pilihan, enter saja untuk memilih To set up Windows XP now, press ENTER.

6. Kemudian akan muncul halaman license agreement, tekan saja tombol F8.

7. Kita asumsikan bahwa hardisk laptop sudah terpartisi biasanya dan seharusnya ada dua partisi, menjadi partisi C dan partisi D. Bila belum baca dulu: cara membuat partisi hardisk baru (cooming soon...).

8. Pilih pada partisi C dengan menggunakan tombol anak panah naik turun pada keywboard.

9. Selanjutnya akan muncul pilihan untuk melakukan format hardisk, pilih saja "format the partision using the NTFS file system, enter dan tunggu loading.

10. Sampai tahap cara instal laptop baru ini, kita tinggal menunggu saja. Akan ada proses format hardisk pada partisi C sampai proses selesai 100 persen, kemudian otomatis dilanjutkan copy file windows dari CD Windows, tunggu sampai proses copy selesai.

11. Saat proses copy file windows ke hardisk selesai, laptop akan otomatis restart, untuk mempercepat nya tekan saja tombol ENTER.

12. Setelah laptop restart, akan muncul kembali pilihan seperti tahap nomor 4, jangan tekan tombol apapun, biarkan saja dan otomatis proses instal laptop akan berjalan, dan kita hanya suruh menunggu sampai tahap selanjutnya.

13. Saat proses copy file Windows sudah selesai, tunggu sampai ada perintah "Regional and Language Option", pilih saja NEXT.

14. Selanjutnya ada pilihan untuk mengatur waktu (jam), pilih di menu dropdown nya : (GMT+7:00 Bangkok, Hanoi Jakarta) dan tekan NEXT.

15. Pada tahap cara instal laptop baru selanjutnya adalah, kita harus memasukkan Product Key, atau biasa kita sebut dengan serial number Windows, masukkan dan tekan NEXT.

16. Tahap selanjutnya adalah memasukkan nama kamu dan nama organisasi kamu, tulis saja terserah dan NEXT.

17. Tahap berikutnya mengisi computer name dan administrator password, abaikan saja atau isi tetapi harus diingat-ingat administrator password nya, NEXT.

18. Kemudian masuk ke menu Networking setting, biarkan saja seperti bawaan aslinya, tekan NEXT.

19. Selanjutnya muncul menu Workgroup or Computer Domain, biarkan saja seperti default Windows, tekan NEXT.

20. Selanjutnya kita tinggal menunggu saja, proses sedang berjalan.

21. Sampai muncul tulisan Display Setting "To improve the appereance... bla bla bla..." tekan saja tombol OK.

22. Muncul lagi tulisan Monitor Setting "Windows adjusted your screen... dst..." tekan tombol OK lagi.

23. Tunggu beberapa saat, dan tratatataa... Windows kita sudah siap sekarang dan proses cara instal laptop baru kita sudah selesai, dan kita bisa untuk memulai instal program lain yang diperlukan.
