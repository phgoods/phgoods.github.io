4. change into passive form tika deleted the message​


Passive Voice

Change into passive form: Tika deleted the message

[Ubah menjadi bentuk pasif: Tika deleted the message]

-----------------------------------------

Jawaban

Bentuk pasif dari kalimat tersebut adalah:

The message was deleted by Dika

--------------------------------------------

Penjelasan

Kalimat pasif di atas diubah menggunakan rumus kalimat pasif simple past tense. Rumusnya:

Subject + was/were + Verb 3 + by + Agent

     Rumus di atas digunakan karena pada mulanya kalimat tersebut menggunakan pola kalimat simple past tense, terlihat dari penggunaan kata kerja bentuk kedua dalam kalimat tersebut, yaitu deleted.

     To be yang digunakan adalah was karena subject-nya adalah sudut pandang ketiga tunggal (The message).

Nilai dari [tex]x + y[/tex] yang memenuhi adalah [tex]x + y = 19[/tex] dengan [tex]x = \frac{19 + \sqrt{17} }{2}[/tex] dan [tex]y = \frac{19 - \sqrt{17}}{2}[/tex] atau [tex]x = \frac{19 - \sqrt{17} }{2}[/tex] dan [tex]y = \frac{19 + \sqrt{17} }{2}[/tex].

Pendahuluan

Materi pada soal adalah materi mengenai manipulasi aljabar. Untuk memanipulasi aljabar, perlu diketahui persamaan-persamaan yang sering digunakan sebagai berikut :

1. [tex](x + y)^{2}[/tex] = [tex]x^{2} + 2xy + y^{2}[/tex]

2. [tex](x - y)^{2}[/tex] = [tex]x^{2} - 2xy + y^{2}[/tex]

3. [tex]x^{2} - y^{2}[/tex] = [tex](x + y)(x - y)[/tex]

4. [tex]x^{n} - y^{n}[/tex] = [tex](x - y)(x^{n-1} + x^{n-2}y + x^{n-3}y^{2} + .... + y^{n-1})[/tex]

5. Untuk n bilangan ganjil, maka berlaku

[tex]x^{n} + y^{n}[/tex] = [tex](x + y)(x^{n-1} - x^{n-2}y + x^{n-3}y^{2} - x^{n-4}y^{3} + ... + y^{n-1})[/tex]

Khusus untuk kasus ini, akan digunakan persamaan nomor 6.

Pembahasan

Diketahui :

[tex]x^{3} + y^{3} = 1957[/tex]  ... (persamaan 1)

[tex](x + y)(x + 1)(y + 1) = 2014[/tex] ... (persamaan 2)

x dan y merupakan bilangan real.

Ditanyakan :

Nilai [tex]x + y[/tex].

Jawaban :

Dengan menggunakan persamaan nomor 6 di bagian pendahuluan, maka dari persamaan 1 didapat

[tex]x^{3} + y^{3} = 1957[/tex]

[tex](x + y)(x^{2} - xy + y^{2}) = 19 \times 103[/tex]

Dari persamaan 2 juga didapat

[tex](x + y)(x + 1)(y + 1) = 2014[/tex]

[tex](x + y)(xy + x + y + 1) = 19 \times 106[/tex]

Dari persamaan diatas, dapat diasumsikan nilai [tex]x + y = 19[/tex] memenuhi. Untuk membuktikan asumsi tersebut, harus dicari nilai dari x dan y sehingga x dan y terbukti merupakan bilangan real. Dari sini juga didapat

[tex](x^{2} - xy + y^{2}) = 103[/tex]  ...  (persamaan 3)

[tex](xy + x + y + 1) = 106[/tex] ... (persamaan 4)

Kurangi persamaan 3 dengan persamaan 4, maka didapat

[tex](x^{2} - xy + y^{2}) - (xy + x + y + 1) = 103 - 106[/tex]

[tex]x^{2} + y^{2} - 2xy - (x + y) - 1 = -3[/tex]

[tex](x - y)^{2} - (x + y) - 1 = -3[/tex]

[tex](x - y)^{2} - 19 - 1 = -3[/tex]

[tex](x - y)^{2} = 17[/tex]

[tex]x - y[/tex] = [tex]\pm \sqrt{17}[/tex]

Maka didapat [tex]x - y = \sqrt{17}[/tex] atau [tex]x - y = - \sqrt{17}[/tex].

1. Untuk [tex]x - y = \sqrt{17}[/tex]

Jumlahkan [tex]x + y = 19[/tex] dengan [tex]x - y = \sqrt{17}[/tex] , maka didapat

[tex]x + y + (x - y) = 19 + \sqrt{17}[/tex]

[tex]2x = 19 + \sqrt{17}[/tex]

[tex]x = \frac{19 + \sqrt{17} }{2}[/tex]

Lalu kurangi [tex]x + y = 19[/tex] dengan [tex]x - y = \sqrt{17}[/tex], maka didapat

[tex]x + y - (x - y) = 19 - \sqrt{17}[/tex]

[tex]2y = 19 - \sqrt{17}[/tex]

[tex]y = \frac{19 - \sqrt{17}}{2}[/tex]

Langkah selanjutnya adalah mengecek apakah nilai [tex]x = \frac{19 + \sqrt{17} }{2}[/tex] dan [tex]y = \frac{19 - \sqrt{17}}{2}[/tex] memenuhi persamaan awal, untuk persamaan 1 didapat

[tex](x + y)(x^{2} - xy + y^{2})[/tex]

[tex]= (\frac{19 + \sqrt{17} }{2} + \frac{19 - \sqrt{17} }{2} )((\frac{19 + \sqrt{17} }{2})^{2} - \frac{19 + \sqrt{17} }{2} . \frac{19 - \sqrt{17} }{2} + (\frac{19 - \sqrt{17} }{2})^{2})[/tex]

[tex]= 19 \times (\frac{378 + 38\sqrt{17} }{4} - \frac{344}{4} + \frac{378 - 38\sqrt{17} }{4} )[/tex]

[tex]= 19 \times 103[/tex]

[tex]= 1957[/tex]

Lalu untuk persamaan 2, didapat

[tex](x + y)(x + 1)(y + 1)[/tex]

[tex]= (\frac{19 + \sqrt{17} }{2} + \frac{19 - \sqrt{17} }{2} )(\frac{19 + \sqrt{17} }{2} + 1)(\frac{19 - \sqrt{17} }{2} + 1)[/tex]

[tex]= 19 \times (\frac{21 + \sqrt{17} }{2} )(\frac{21 - \sqrt{17} }{2} )[/tex]

[tex]= 2014[/tex]

Karena memenuhi untuk kedua persamaan, maka [tex]x = \frac{19 + \sqrt{17} }{2}[/tex] dan [tex]y = \frac{19 - \sqrt{17}}{2}[/tex] memenuhi.

2. Untuk  [tex]x - y = -\sqrt{17}[/tex]

Jumlahkan [tex]x + y = 19[/tex] dengan [tex]x - y = -\sqrt{17}[/tex] maka didapat

[tex]x + y + (x - y) = 19 + (-\sqrt{17} )[/tex]

[tex]2x = 19 - \sqrt{17}[/tex]

[tex]x = \frac{19 - \sqrt{17} }{2}[/tex]

Kemudian kurangi [tex]x + y = 19[/tex] dengan [tex]x - y = -\sqrt{17}[/tex] maka didapat

[tex]x + y - (x - y) = 19 - (-\sqrt{17} )[/tex]

[tex]2y = 19 + \sqrt{17}[/tex]

[tex]y = \frac{19 + \sqrt{17} }{2}[/tex]

Perhatikan bahwa nilai x dan y yang didapatkan merupakan hasil tukar nilai x dan y pada pengecekan pertama. Karena [tex](x + y)(x^{2} - xy + y^{2})[/tex]  = [tex](y + x)(y^{2} - yx + x^{2})[/tex] dan karena [tex](x + y)(x + 1)(y + 1)[/tex] = [tex](y + x)(y + 1)(x + 1)[/tex], maka hasilnya sama saja dengan pengecekan 1 sehingga untuk [tex]x = \frac{19 - \sqrt{17} }{2}[/tex] dan [tex]y = \frac{19 + \sqrt{17} }{2}[/tex] memenuhi kedua persamaan diatas.

Karena untuk [tex]x + y = 19[/tex] menghasilkan x dan y bilangan real, maka jawabannya adalah [tex]x + y = 19[/tex].
