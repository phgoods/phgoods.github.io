Jawaban:

Misalkan dipakai pada program pengolah data teks (seperti Ms. Word):

Caps Lock: untuk mengunci huruf kapital.

Tab: untuk memindah posisi kursor ke sebelah kanan 1 tabulasi.

Back space: untuk menghapus 1 karakter di sebelah kiri

Delete : untuk menghapus 1 karakter di sebelah kanan

Enter: untuk mengakhiri 1 baris dan mengawali pengetikan teks di baris baru (di bawahnya).

Esc: untuk menghilangkan tampilan menu yg muncul.

Home: untuk memindah posisi kursor ke awal baris teks.

End: untuk memindah posisi kursor ke akhir baris.

Page Up: untuk memindahkan posisi kursor 1 layar ke atas.

Page Down: untuk memindahkan posisi kursor 1 layar ke bawah.

Penjelasan:

Contoh pemakaian tombol Esc: misalkan mengklik kanan teks (dg mouse) lalu muncul suatu menu, maka tombol ini bisa dipakai untuk menghilangkan menu yg muncul tadi.

Jawaban:

21. apa langkah langkah ketika membuat tombol back


Menggunaan aplikasi Back Button

1. Langkah pertama tentunya kita perlu mengunduh dan memasang aplikasinya terlebih dahulu. Aplikasi yang akan kita gunakan bernama Back Button yang dapat diunduh di Google Play.

￼

2. Setelah aplikasi terpasang, jalankan aplikasi, kita akan melihat tampilan untuk mengaktifkan ijin akses, pilih Enable untuk mengaktifkan ijin akses.

￼

3. Setelah ijin akses di aktifkan kita akan melihat aplikasi. Setelah itu gesek kearah kanan pada tombol “Active” untuk mengaktifkan aplikasi.

￼

4. Setelah itu kembali buka aplikasi Back Button karena kita perlu mengaktifkan tombol apa saja yang kita inginkan muncul. Pilih tombol yang kita tampilkan, maka kita akan melihat tombol tersebut di perangkat Android.

￼

Untuk mengatur posisinya, cukup lakukan ketukan pada tombol tersebut. Selain itu kita juga dapat mengatur ukuran dan melakukan perubahan-perubahan lain sesuai kebutuhan kita.

27. Jelaskan fungsi dari tombol back, forward, home


Pada aplikasi web browser (Mozilla Firefox, Chrome, Internet Explorer, dsb),
- Tombol Back berfungsi untuk kembali ke halaman web sebelumnya. Misal kita sedang ada di halaman 2 suatu web, jika ingin kembali ke halaman 1, tinggal klik tombol Back
- Tombol Forward berfungsi untuk maju ke halaman web selanjutnya. Misal jika kita sudah ada di halaman 1 (masih berkaitan dengan contoh yg sebelumnya), dan ingin balik lagi ke halaman 2 tinggal klik tombol Forward 
- Tombol Home berfungsi untuk kembali ke Homepage/halaman awal dari web browser tersebut
