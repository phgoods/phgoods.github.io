Hasil dari [tex] \sf 7! + 6! × 5! [/tex] adalah 91.440

Pendahuluan :

Faktorial merupakan merupakan penghitungan perkalian yang banyaknya susunan. Faktorial disebut dengan n!, Dilambangkan dengan n. Sehingga rumus Faktorial adalah n! = n × (n - 1) × (n - 2) × … × 1 atau juga n! = n × (n - 1).

Kaidah pencacahan merupakan Cabang Matematika tentang membahas aturan menghitung banyaknya susunan atau kombinasi sesuatu objek tanpa ada harus merinci semua kemungkinan susunannya. Aturan Kaidah pencacahan ada 4 jenis yaitu ada, Perkalian, Faktorial, Kombinasi, dan Permutasi.

Pembahasan :

[tex] \sf7! + 6! × 5![/tex]

[tex] \sf = 7! + 6! × 5![/tex]

[tex] \sf =5.040 + 6! × 5![/tex]

[tex] \sf = 5.040+ 720× 5![/tex]

[tex] \sf = 5.040 + 86.400[/tex]

[tex] \sf = 91.440[/tex]

1. Jadi, Hasil dari [tex] \sf 7! + 6! × 5! [/tex] adalah 91.440

12. Dalam c++ Input sebuah angka : Bila angka mod 2 = 0 maka cetak bilangan genap 2,4,6,......50 Bila angka mod 2 = 1 maka cetak bilangan ganjil 1,3,5,.....100 Buat lah dalam c++


berikut adalah kode dalam bahasa C++

#include <iostream>

using namespace std;

int main( ){

int n;

cin>>n;

if( n % 2 == 0 ){

cout << "Bilangan Genap : 2";

for( int i = 4; i <= 50; i += 2 ){

cout<<", "<<i;

}

cout<<endl;

} else {

cout<< "Bilangan Ganjil : 1";

for( int i = 3; i < 100; i += 2 ){

cout<<", "<<i;

}

cout<<endl;

}

return 0;

}

PEMBAHASAN

Pertama, kita perlu membaca input suatu angka. Gunakan syntax if-else untuk menentukan sebuah kondisi.

Kondisi 1 : Jika angka tersebut habis dibagi 2, maka program akan mencetak bilangan genap antara angka 2 dan 50 inklusif. Untuk mempermudah proses pencetakan, kita gunakan syntax for dengan nilai awal adalah 4, nilai maksimum adalah 50, dan perubahan nya adalah ditambah 2.Kondisi 2 : Jika angka tersebut menyisakan 1 jika dibagi 2, maka program akan mencetak bilangan ganjil antara angka 1 dan 100. Untuk mempermudah proses pencetakan, kita gunakan syntax for dengan nilai awal adalah 3, nilai maksimum adalah 99, dan perubahannya adalah ditambah 2.

Pada C++, Penulisan If-else adalah

if ( <kondisi> ) {

<perintah 1>;

<perintah 2 >;

...

} else {

<perintah a>;

<perintah b>;

...

}

<perintah 1>, <perintah 2>, ... dalam if akan dijalankan jika kondisi bernilai true, sebaliknya jika kondisi bernilai false, maka <perintah a>, <perintah b>, ... akan dijalankan

Pada C++, penulisan for adalah

for( <inisiasi awal>; <inisiasi akhir>; <perubahan> ){

<perintah 1>;

<perintah 2>;

...

}

selama nilai dari inisasi awal belum sama dengan inisiasi akhir <perintah 1>, <perintah 2>, ... akan terus dijalankan sampai nilai dari inisiasi akhir sama dengan nilai dari inisiasi awal.

19. apa verb 2 dari drink? plssssss bantuin nanti ku follow dan ada yg tau gacha life


Verb 2 dari drink adalah drank

Pembahasan

Verb, adalah kata kerja dari suatu tindakan. Verb dibagi menjadi beberapa macam yaitu

Verb 1 (Kata awal)

Contohnya : Drink

Contoh kalimatnya : I drink a glass of milk

Verb 2 (Kata lampau/masa lalu)

Contohnya : Drink => Drank

Contoh kalimatnya : I drank a cup of coffee yesterday

Verb 3 (Kata kerja tak beraturan)

Contohnya : Drink => Drank => Drunk

Contoh kalimatnya : They have drunk a cup of tea made from me

Subject verb  

Verb 1 dengan Diikuti dengan subjek, misalkan he, she, it

Contohnya : Drink => Drinks

Contoh kalimatnya : He drinks something

Verb -ing

Verb 1, tapi diikuti dengan -ing

Contohnya : Drink => Drinking

Contoh Kalimatnya : She always drinking milk in the morning
