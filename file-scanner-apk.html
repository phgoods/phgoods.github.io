22. format file hasil penyimpanan scanner dengan standar Windows format​


Jawaban:

PDF (Portable Document Format)

Format penyimpanan file dalam bentuk “.pdf” yang mampu menampilkan data berupa teks maupun gambar. Setiap file menyimpan bentuk format penulisan dan jenis huruf. Secara umum dibuka menggunakan software Adobe Acrobat Reader.

JPEG (Joint Photography Expert Group) atau JPG

Format ini dikhususkan untuk penyimpanan file gambar/foto. Format “.jpeg” ini dapat diatur tingkat kompresi nya, sehingga antara kualitas dan kapasitas penyimpanan bisa diatur (semakin besar kompresi gambar maka semakin kecil ukuran file, begitu juga sebaliknya). Akan tetapi, hasil scan JPEG apabila dikompres terlalu banyak akan menghilangkan detail gambar dan warna yang kurang tajam. Secara umum dibuka menggunakan Windows Picture, Paint, dan software lain yang menunjang.

PNG (Portable Network Graphic)

Format file “.png” ini dikhususkan juga untuk penyimpanan file gambar/foto, hampir sama seperti format .jpeg. Namun bentuk file ini optimal untuk menyimpan gambar yang banyak menggunakan warna solid (tidak lebih dari 256 warna), karena dasar algoritma penyimpanannya menggunakan “jumlah warna” hasil penyimpanannya hampir tanpa kompresi sama sekali, sehingga file gambar akan tetap sama seperti aslinya. Bentuk format ini bisa menampilkan gambar transparan. Secara umum dibuka menggunakan software Windows Picture, Paint, dan software lain yang menunjang.

BMP (Bitmap)

Hampir sama dengan format .tiff hanya bedanya “.bmp” biasanya digunakan untuk format file uncompressed (tidak ada kompresi) di sistem operasi Windows. Akan tetapi format ini sudah jarang digunakan karena ukuran file yang besar, sedangkan kualitasnya bisa hampir disamakan dengan format lainnya. Kelebihannya adalah file ini mendukung transparant layer. Secara umum dibuka menggunakan software Windows Picture, Paint, dan software lain yang menunjang.

Kecepatan prosesor dan RAM dikukur melalui kemampuan clock rate atau juga biasa disebut dengan clock speed, yaitu kecepatan dari kedua bagian komputer tersebut dalam mengeksekusi setiap instruksi yang diperintahkan. Setiap komputer sendiri memiliki internal clock yang berfungsi untuk mengatur serta menyinkronkan komponen-komponen yang ada.

Pembahasan:

Pada umumnya sebuah komputer terdiri dari Central Processing Unit (CPU yang kita kenal dengan prosesor utama) dan mikroprosesor, yaitu prosesor lain yang terpasang pada komponen diluar CPU. Misalnya RAM dan sirkuit kartu grafis. Dalam hal ini komponen diluar CPU yang memiliki mikroprosesor akan memiliki clock rate seperti halnya prosesor utama.

Clock rate sendiri memiliki satuan unit hertz (Hz) Untuk generasi pertama rata-rata masih menggunakan kilohertz (kHz) lalu berkembang menjadi megahertz (MHz) hingga pada generasi komputer yang ada pada saat ini yang sudah mencapai ukuran Gigahertz (GHz) Jadi sudah bisa dibayangkan bukan pesatnya perkembangan clock speed dari komputer pertama hingga sekarang?

Soal akses sendiri tergantung dari komputer itu sendiri tapi pada dasarnya setiap komputer sudah memiliki kecepatan maksimum yang ditetapkan. Nah, kalau kamu ingin mengubah kecepatan maksimum yang sudah ada maka kamu harus mengaksesnya melalui BIOS. AKtivitas meningkatkan kecepatan clock speed ini dikenal dengan sebutan overclocking.

Untuk melihat kecepatan prosesor yang digunakan kamu bisa menggunakan bantuan aplikasi CPU-Z. Sebagai contoh saya lampirkan screenshot tampilannya.

30. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.
