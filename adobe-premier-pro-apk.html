17. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.

26. jelaskan fungsi dari program adobe premier pro



Fungsi utama Premiere Pro lebih efektif lagi untuk merangkai gambar, video dan audio, bukan untuk animasi. Agar penampilan multimedia anda lebih menarik, sebaiknya dipelajari pula software animasi dan grafis lain seperti 3D Studio Max, After Effects, Adobe Photoshop dan utility multimedia lainnya.
Tool pada Adobe Premiere
Toolbox berisi alat-alat yang digunakan untuk mengedit klip pada timeline. Klik pada salah satu tombol (atau menggunakan cara pintas pada keyboard) untuk memilih setiap alat. Alat default adalah alat seleksi pada time line.
Bila kita memilih alat pointer mouse biasanya akan berubah menjadi sebuah ikon baru untuk mewakili alat ketika selama kursor pada panel timeline.
Dalam beberapa kasus, anda dapat mengubah fungsi alat dengan menekan tombol pengubah seperti tombol Shift.
