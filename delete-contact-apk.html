13. Jelaskan perbedaan fungsi delete entire row dengan delete entire column


Sebelum menjawab pertanyaan mari pahami terlebih dahulu pengertian dari opsi Delete entire. Delete entire merupakan sebuah cara untuk menghapus suatu data dalam satu rentang data baik baris maupun kolom dalam software Microsoft Excel.

Dibawah ini adalah penjelasan antara fungsi delete entire row dan delete entire column:

1. Delete Entire Row

Opsi entire row digunakan saat akan menghapus data di excel, satu atau lebih baris dari ujung kiri sampai ujung kanan. Jadi pada saat memilih opsi ini saat melakukan penghapusan data, semua data yang ada pada baris / rentang baris yang dipilih akan dihapus.

Kelebihan dari delete entire row ini adalah dapat menghapus rentetan data dalam suatu baris jika sudah tidak digunakan lagi secara sekaligus.


2. Delete Entire Columns

Opsi entire column ini hampir sama dengan cara delete menggunakan entire row. Hanya saja pada pilihan ini, penghapusan data dilakukan pada kolom dari ujung atas sampai ujung bawah.

Penggunaan opsi delete entire column ini akan dapat menghapus suatu data atau sekelompok data yang sudah tidak digunakan dalam data bentuk vertical.

23. jelaskan perbedaan delete entire row dengan delete entire column


Delete Entry Row : menghapus baris
Delete Entry Coloum : menghapus kolom

Contoh yang Row (Baris) :
Dikolom A, ada 4 baris data,
1. AA1
2. AA2
3. AA3
4. AA4
Kita memilih data nomor 3 untuk di Delete Entry Row, setelah didelete, data nomor 3 akan hilang, dan hasilnya akan menjadi :
1. AA1
2. AA2
4. AA4

----------

Kalau Coloum :
Di Colom A, B, C, D, E ada beberapa data.
Nah kita memlilih Delete Entry Row di Kolom C, maka semua data di Kolom C, akan Hilang.

----------

Untuk membedakan antara Baris dan Kolom :
Untuk Baris, selalu "Atas Bawah"
Untuk Kolom, selalu "Kesamping" / Kekiri, atau Ke Kanan

Jawaban:

Sintaks SQL untuk mendelete record adalah D. Delete from tabel where field1= " ". Kesalahan pada opsi A adalah tidak adanya penggunaan "WHERE" dalam sintax akan menghapus record tanpa menghapus tabel. Sedangkan opsi B akan menjadi sintaks yang menghapus tabel bukan record. Lalu pada opsi C, sintaks akan error karena kondisi yang tidak jelas.

Penjelasan:

Format penulisan sintaks SQL untuk mendelete record adalah

DELETE FROM nama tabel WHERE kondisi; . Bagian vital dalam perintah ini adalah penggunaan kata WHERE. Apabila WHERE terlewatkan untuk memasukkan kondisi, maka seluruh record justru terhapus. Jika ingin menghapus seluruh record tanpa menghilangkan struktur tabel, dapat digunakan perintah DELETE FROM  nama tabel ;
