1. Apa itu ciri ciri frame rate


Frame rate (dinyatakan dalam frame per second atau fps ) adalah frekuensi (rate) di mana gambar berturut-turut disebut frames muncul pada layar. Istilah ini berlaku sama untuk kamera film dan video , grafis komputer, dan sistem pengambilan gambar . Frame rate juga bisa disebut frekuensi frame , dan dinyatakan dalam hertz .

Intinya ciri-ciri adanya frame rate itu seperti ada berkas-berkas gambar/ jejak-jejak gambar.

Ini adalah materi lanjutan dari Action Script yang terdapat pada Macromedia Flash 8. Jika pada materi sebelumnya, pembahasannya adalah action script general, maka kali ini saya akan menjelaskan tentang action script movie atau frame pada Macromedia Flash 8.

Beberapa command yang saya pelajari yaitu “Play”, “Stop”, “Next Frame”, “Previous Frame”, “Go to And Play”, dan “Go to And Stop”. Penggunaan Action script ini biasanya digunakan pada tombol atau button pada animasi atau project yang dibuat.

Play

Command “play()” pada objek tombol berfungsi untuk mulai memainkan suatu animasi.

Contoh penulisan command :

on(release)

{

play();

}

Stop

Command “stop()” pada objek tombol berfungsi untuk menghentikan animasi yang dijalankan.

Contoh penulisan command :

on(release)

{

stop();

}

Previous Frame

Command “prevFrame()” pada objek tombol berfungsi untuk menghentikan animasi yang dijalankan lalu kembali ke posisi 1 frame sebelumnya. Animasi akan berjalan mundur frame per frame sebanyak 1 frame tiap kali tombol dengan command “prevFrame()” ditekan.

Contoh penulisan command :

on(release)

{

prevFrame();

}

Next Frame

Command “nextFrame()” pada objek tombol berfungsi untuk menghentikan animasi yang dijalankan lalu maju ke posisi 1 frame selanjutnya. Animasi akan berjalan maju frame per frame sebanyak 1 frame tiap kali tombol dengan command “nextFrame()” ditekan.

Contoh penulisan command :

on(release)

{

nextFrame();

}

Go to And Play

Command “gotoAndPlay()” pada objek tombol berfungsi untuk melanjutkan animasi atau project yang sedang berjalan, menuju ke frame tertentu yang telah ditentukan. Dan animasi akan langsung berjalan secara otomatis. Masukkan frame tujuan didalam tanda kurung.

Contoh penulisan command dengan frame tujuan frame ke-50 :

on(release)

{

gotoAndPlay(50);

}

Go to And Stop

Command “gotoAndStop()” pada objek tombol berfungsi untuk melanjutkan animasi atau project yang sedang berjalan, menuju ke frame tertentu yang telah ditentukan. Tetapi animasi tidak akan berjalan secara otomatis, melainkan akan berhenti pada frame yang telah ditentukan tadi. Command ini biasanya digunakan untuk kembali ke menu utama project. Masukkan frame tujuan didalam tanda kurung.

Contoh penulisan command dengan frame tujuan frame ke-10 :

on(release)

{

gotoAndStop(10);

}

13. Diketahui file video dengan waktu 30 menit, dengan 24 bit, dan Frame rate 30 FPS, dan Resolusi 800 x 600 , Pertanyaannya berapa besar file video tersebut ?


Jawaban:

Besar file video tersebut adalah 72,42 Gb

Penjelasan:

1) Variabel yang diketahui

Sebelum mengerjakan soal ini, ada beberapa variabel yang perlu diketahui, yaitu durasi video (dalam detik/sekon), bit depth, frame rate, dan resolusi (panjang dan lebar video).

Dari soal, dapat diketahui:

durasi = 30 menit x 60 detik = 1800 detik

bit depth =  24 bit

frame rate = 30 FPS

panjang video = 800 pixels

lebar video = 600 pixels

2) Rumus/formula yang digunakan

Ada 3 (tiga) rumus yang akan digunakan di sini, yaitu:

a. Jumlah frame

= Durasi (sekon) × frame rate

b. Ukuran file per frame

= [tex]\frac{Horizontal.Vertikal.Bit Depth}{8 . 1024}[/tex]

c. Ukuran video

= Jumlah frame × Ukuran file per frame

3) Masukkan variabel ke dalam rumus

a. Jumlah frame

= 1800 detik × 30 FPS

= 54.000 frames

b. Ukuran file per frame

= [tex]\frac{800.600.24}{8.1024}[/tex]

= 1406,25 Kb

c. Ukuran video

= 54.000 frames x 1406,25 Kb

= 75.937.500 Kb

= 74.157,7 Mb (dibagi 1024)

= 72,42 Gb (dibagi 1024 lagi)

24. high speed video can record sporting events at a frame rate of 60 frames per second. what is the time interval between one frame and the next. b. if we can see 24 frame as continuous motion, by what factor can be the action recorded at 60 frame be slowed down and still look continuous


Jika sebuah video berkecepatan tinggi dapat merekam acara olahraga dengan frame rate 60 frame per detik, maka interval waktu antara satu frame dan frame berikutnya adalah 1/60 detik. Jika kita dapat melihat 24 frame sebagai gerakan yang terus-menerus, maka gerakan yang direkam dengan frame rate 60 frame dapat diperlambat sebanyak 60/24 = 2,5 kali dan masih terlihat sebagai gerakan yang terus-menerus. Namun, perlu diingat bahwa hal ini hanya merupakan perkiraan dan tidak dapat digunakan secara akurat untuk menentukan seberapa jauh sebuah video dapat diperlambat tanpa kehilangan efek gerakan yang terus-menerus. Hal ini bergantung pada banyak faktor, seperti kecepatan dan kompleksitas gerakan yang direkam, serta preferensi individu yang melihat video tersebut. Jadi, hasil perhitungan ini hanya merupakan perkiraan saja dan tidak dapat digunakan secara akurat dalam situasi nyata.
