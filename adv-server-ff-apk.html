6. 10 kalimat yang menggunakan as+adv+as​


As + adv + as adalah pola dalam pembuatan positive degree atau perbandingan tingkat sebanding.

Berikut adalah beberapa contoh.

1. Rina's height is 170 cm. Mita's height is 170 cm. Rina is as tall as Mita.

2. Dodi's house is 1 km from school. Gita's house is 1 km from school. Dodi's house is as far as Gita's house from school.

3. Titin got 100 in Math. Vina got 100 in Math. Titin is as smart as Vina in Math.

4. Everybody knows Rian. Everybody knows Vian. Rian is as famous as Vian.

5. Lili is a beautiful girl. Her twin, Lila, is a beautiful girl. Lili is as beautiful as Lila.

6. Todi can run 10m in one second. Dion can run 10m in one second. Todi is as fast as Dion.

7. Jon never wanted to study. Rino never wanted to study. Jon is as lazy as Rino.

8. Tina is clumsy. Mina is clumsy. Tina is as clumsly as Mina.

9. Hulk is strong. Superman is strong. Hulk is as strong as Superman.

10. Thor is brave. Spiderman is brave. Thor is as brave as Spiderman.

Adjective is one part of sentence which states about a quality of something. Examples of verbs are smart, thin, good, bad, inquisitive, popular etc.
Adverb or adv is apart of the sentence  as words which tells the way of doing something or states the frequency of doing something. Here's the example its regularly, rarely, occasionally, well, badly, carefully

14. apa perbedaan adv of manner adv of time dan adv of frequency ??


Adverb of manner itu seperti : diligently, lazily, intensively.
Semacam keterangan sifat.

Adverb of time itu seperti : monthly, annually, daily.
Semacam keterangan waktu.

Adverb of frequency itu seperti : usually, often, seldom.
Semacam keterangan frekuensi.

Semoga membantu xD.adv of manner adl kata keterangan yg digunakan utk menerangkan suatu kejadian tjd spt apa. contoh: he is driving carefully.

adverb of time adl kata keterangan yg menunjukkan waktu suatu kejadian.
contoh. he drove yesterday.

adverb of frequency adl kata keterangan yg menerangkan berapa sering kejadian tsb dilakukan.
contoh: he usually plays football every sunday morning.
(every sunday morning = adv of time, usually = adv of frequency)

= 5! - 3!

Karena tanda (!) adalah Faktorisasi bilangan dengan operasi hitung Perkalian (×).

Jadi 5! - 3! bukan 2! tapi 114

#CMIIW----Notasi Faktorial-----

Untuk bilangan n bilangan asli

Faktorial dapat dilambangkan dengan (!)

n! = 1 x 2 x 3 x ... x (n-1) x n

n! = n x (n-1)

Ditanyakan

5! - 3! = 2!

apakah hasil dari pertanyaan itu benar? jelaskan

Penyelesaian

5! - 3! = 2!

(5x4x3x2!) - (3x2!) = 2!

120 - 6 = 2

114 = 4

Hasil dari pernyataan tersebut salah karena 114 tidak sama dengan 4

19. Server Adalah web server adalah Server FTP adalah Server DHCP Adalah


Mapel : TI
Materi : Server

--Soal--

Server Adalah
Web server adalah
Server FTP adalah
Server DHCP Adalah

--Jawaban--

Server adalah suatu sistem komputer yang menawarkan jenis layanan tertentu dalam suatu jaringan.

Web server adalah suatu perangkat lunak (software) yang memberikan perintah berbasis data yang berfungsi menerima permintaan dari http atau https pada klien web terkenal.

Server FTP adalah protokol internet yang berjalan dalam suatu lapisan aplikasi yang berfungsi untuk tukar menukar berbagai data antara klien dan server dalam 1 jaringan.

Server DHCP adalah adalah suatu protocol pada jaringan komputer yang dapat memberikan atau meminjamkan atau memberi akses IP address terhadap host yang berada dalam suatu jaringan secara otomatis.
