4. Tau cara membuka passcode di iphone ga kalo lupa pass


Kalo passcode di iphone ke lock dan lupa sih biasanya bawa ke counter hp atau ke ibox nya langsung
maaf kalo salahCara Reset iPhone ke Factory Default dengan Recovery Mode
- Matikan iPhone dengan cara menekan tombol power
- Jalankan iTunes di Laptop
- Sambungkan iPhone ke Laptop, tekan dan tahan tombol Home dan tombol Power secara bersamaan selama 10 detik. Setelah 10 detik lepaskan tombol power sampai iTunes mendetect iPhone dalam Recovery Mode. Setelah iTunes mendetect iPhone dalam Recovery Mode, baru Anda bisa melepas tombol Home.
- Selanjutnya masuk ke iTunes dan pilih Restore untuk merestore iPhone ke Factory Default, proses ini membutuhkan koneksi internet.
- Jika sebelumnya Anda sudah mendownload Firmware iPhone, Anda bisa melakukan restore secara offline. Caranya tekan dan tahan tombol SHIFT di keyboard laptop, kemudian klik Restore, sehingga nanti akan ada pilihan untuk memilih (browser) file Firmware iPhone yang sudah Anda sebelumnya.



Sebelumnya perlu diingat, bahwa reset passcode iPhone bisa dilakukan dengan cara reset iPhone ke Factory Default, dengan resiko semua data di iPhone hilang. Namun Anda jangan khawatir, jika Anda sering melakukan backup di iTunes, maka nantinya data bisa di restore.

12. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.

5. Pilih file presentasi yang diinginkan, lalu klik open
