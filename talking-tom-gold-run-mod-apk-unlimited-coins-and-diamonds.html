18. christy has several $5 coins and $2 coins. it is given the total value of the coins is $38. if she uses two $5 coins to buy a pen, then the numbers of $5 coins and $2 coins will be the same. find the original number of coins


Jawaban:

The original number of $5 coins is 6.The original number of $2 coins is 4.

Penjelasan dengan langkah-langkah:

Number of $5 coins = x

Number of $2 coins = y

Total value is $38:

5x + 2y = 38 ... (1)

If two $5 coins used, then the number of $5 and $2 coins will be the same:

y = x - 2 ... (2)
