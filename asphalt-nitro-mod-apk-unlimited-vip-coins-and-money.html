24. Helen and Ivan had the same number of coins. Helen had a number of 50-cent coins and 64 20-cent coins. These coins had a mass of 1.134kg. Ivan had a number of 50-cents coins and 104 20-cent coins. A) given that each 50-cent coin is 2.7g more heavier than a 20-cent coin, what is the mass of Ivan's coin in kg?​


Jawab:

1.026 kg

Penjelasan dengan langkah-langkah:

Let n be the number of Helen's 50-cent coins.

It is given that:

Helen and Ivan had the same number of coinsHelen had a number of 50-cent coins and 64 20-cent coinsIvan had a number of 50-cents coins and 104 20-cent coins

104 – 64 = 40

From these facts, we know that Ivan had 40 20-cent coins more than Helen's. Therefore, we can get that the number of Ivan's 50-cent coins which equals to:

n – 40

Let x be the mass of each 50-cent coin, and y be the mass of each 20-cent coin.

At Helen's side, we can get that

nx + 64y = 1.134kg = (1.134 * 1000) g

⇔  nx + 64y = 1,134     .....(i) (in grams)

Given that each 50-cent coin is 2.7g heavier than a 20-cent coin, we can get that

x = y + 2.7      (in grams)

Substitute x to equation (i):

n(y + 2.7) + 64y = 1,134     .....(ii)

Let m be the mass of Ivan's coins, we can get that:

(n – 40)x + 104y = m

⇔  nx – 40x + 104y = m

Then we substitute x with y + 2.7.

⇔  n(y + 2.7) – 40(y + 2.7) + 104y = m

⇔  n(y + 2.7) – 40y – 40(2.7) + 104y = m

⇔  n(y + 2.7) + 64y – 108 = m

Substitute n(y + 2.7) + 64y with 1,134 we got in equation (ii):

⇔  1,134 – 108 = m

⇔  m = 1,026 g = (1,026 / 1,000) kg

⇔ m = 1.026 kg

CONCLUSION:

∴  The mass of Ivan's coins is 1.026 kg.

Penjelasan dengan langkah-langkah:

From what we have, we will say that Kelly has an x amount of coins, that means Linda has 3x coins. and Jo has x + 362. Let's list that down.

Jo: x + 362

Linda: 3x

Kelly: x

we can work from that, that means if we add it all together that has a result of:

5x + 362 = 5102

Now, we move the total around,

5102 = 362 + 5x

5102 - 362 = 4740

4740 = 5x

5x = 4740

4740 ÷ 5 = 948

x = 948

Now we know how much Kelly has, now we add it up.

x + 362 = 1310

So,

Jo has 1310 Coins.
