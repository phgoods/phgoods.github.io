Persamaan kalor untuk menaikkan suhu benda:

Kalor yang diperlukan untuk menaikkan suhu benda bergantung pada jenis bendanya. Semakin besar kenaikan suhu benda, kalor yang diperlukan semakin besar, dan semakin besar massa benda, kalor yang dibutuhkan pun semakin besar. Secara matematis, persamaan kalor untuk menaikkan suhu benda dirumuskan sebagai berikut.

Q = m x c x ΔT

dimana:

Q = kalor yang diterima/dilepas suatu benda (J)

m = massa benda (kg)

c = kalor jenis (straight J/kg°C)

ΔT = perubahan suhu (°C)

Persamaan kalor untuk mengubah wujud benda :

Kalor yang digunakan untuk mengubah wujud zat disebut dengan kalor laten. Kalor laten terdiri dari kalor penguapan/pengembunan dan kalo lebur/beku.

Kalor penguapan/pengembunan

Q = m x U

dimana:

Q = kalor yang diperlukan/dilepas untuk berubah wujud (J)

m = massa zat yang berubah wujud (kg)

U = kalor penguapan/kalor pengembunan (J/kg)

Kalor lebur/beku

Q = m x L

dimana:

Q = kalor yang diperlukan/dilepas untuk berubah wujud (J)

m = massa zat yang berubah wujud (kg)

L = kalor lebur/kalor beku (J/kg)

[tex]\frac{1}{30}[/tex] x [tex]\frac{1}{2}[/tex] = [tex]\frac{1 . 1}{30 . 2}[/tex]

[tex]\frac{1}{30}[/tex] x [tex]\frac{1}{2}[/tex] = [tex]\frac{1}{60}[/tex]

23. tolong bantu kk, . latihan soal tiu cpns


Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)
