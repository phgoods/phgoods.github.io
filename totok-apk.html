28. jika selisih uang Andi dan Totok = 3:2, sedangkan Totok dan Dani = 4:5. Jumlah uang Andi dan Dani 44.000, maka Totok adalah...

Bab : Perbandingan

Diketahui :
- Andi : Totok = 3 : 2
- Totok : Dani = 4 : 5
- Jumlah Uang Andi dan Dani = 44.000

Ditanya :
- Uang Totok = ......?

Penyelesaian :
- Perbandingan :
Andi : Totok = 3 : 2
Totok : Dani = 4 : 5
———————————————
Andi : Totok : Dani = 6 : 4 : 5

- Uang Totok :
4/(6 + 5) × 44.000
= 4/11 × 44.000
= 4 × 4.000
= 16.000
