2. Fitur auto update

Berhasil Mengatasi Download Tertunda di Play Store

Seperti yang sudah aku singgung di tips pertama, adanya fitur auto update di Play store ini bisa menjadi penyebab download tertunda

Sebagai gambaran begini, ketika kamu masih mengaktifkan fitur auto update. Dan kamu terhubung ke WiFi maka otomatis aplikasi akan update

Dan kamu mau download baru maka akan tertunda menunggu dulu aplikasi atau game yang update otomatis tadi selesai. Jadi ini hampir mirip seperti masalah pertama yang download bersamaan

Kamu bisa cancel updatenya dan matikan fitur auto update di Play Store, selain bisa mengatasi internet HP android lemot mematikan auto update ini juga bisa mengurangi penggunaan memory, jadi HP memory tidak cepat penuh

Buka google play store

Pilih setting

Pada menu update otomatis rubah ke jangan update otomatis

Silahkan coba download ulang apliaksi atau game yang tadinya muncul download tertunda, jika masih gagal update versi google playstore

4. Update google playstore

Pastikan kamu menggunakan versi Google Playstore yang terbaru agar semua fitur dan juga bug atau kerusakan bisa tertutup

Untuk updatenya cukup gampang, pastikan internet lancar dan silahkan buka aplikasi Google Playstore

4. cache browser adalah


Cache browser, kadang-kadang juga disebut sebagai folder file sementara Internet, berisi file dari situs web yang Anda kunjungi. Semua web browser modern menjaga file cache untuk satu alasan penting, tujuan nya adalah untuk menampilkan halaman web lebih cepat di waktu berikutnya apabila Anda membuka lagi situs-situs web tersebut. Alasan nya karena file halaman web bukan diambil dari server web online tetapi diambil dari cache browser (di komputer Anda) sehingga halaman web dapat tampil lebih cepat.
Cache Browser Internet adalah sama dengan folder Temporary Internet Files, dimana kumpulan catatan-catatan atau record dari riwayat jejak browsing yang anda lakukan pada browser anda.
