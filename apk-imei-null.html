4. contoh query update agar bisa menampung nilai NULL​


Untuk mengupdate sebuah record di tabel database dengan nilai NULL, kamu dapat menggunakan perintah `UPDATE` seperti ini:

UPDATE tabel SET kolom = NULL WHERE kondisi;

Contoh:

UPDATE users SET email = NULL WHERE id = 1;

Ini akan mengupdate kolom email untuk record dengan id 1 dengan nilai NULL.

Perhatikan bahwa jika kamu menggunakan tanda petik tunggal (') atau tanda kutip (") untuk menyatakan nilai NULL, maka akan dianggap sebagai string, bukan sebagai nilai NULL. Jadi pastikan untuk tidak menggunakan tanda petik apapun saat menyatakan nilai NULL.

Jika kamu menggunakan salah satu dari bahasa pemrograman yang menggunakan tanda petik tunggal atau tanda kutip sebagai tanda string, maka Anda harus menggunakan tanda backslash () untuk mengabaikan tanda petik tersebut, seperti ini:

UPDATE users SET email = '\'NULL\'' WHERE id = 1;

Atau, kamu dapat menggunakan tanda petik ganda (") untuk menyatakan nilai NULL, seperti ini:

UPDATE users SET email = "NULL" WHERE id = 1;

Namun, lebih disarankan untuk tidak menggunakan tanda petik sama sekali dan hanya menggunakan NULL saja.

JAWABAN :

Interval prime ditunjukkan pada nomor 1 ( do - do )

Pembahasan :

1. Prim: yaitu interval nada dari nada satu ke nada yang sama. Misal: dari nada do ke do

2. Sekon: yaitu interval nada dari nada satu ke nada kedua di atas atau di bawahnya. Misal nada do ke re

3. Terts: yaitu interval nada dari nada satu ke nada ketiga: Misal nada do ke mi

4. Quart/Kuart: yaitu interval dari nada kesatu ke nada keempat di atasnya. Misal nada do ke fa, re ke sol, mi ke la, dan sebagainya.

5. Quin/Kuint: adalah interval lima nada.

6. Sekt: adalah interval enam nada.

7. Septim: adalah interval tujuh nada.

8. Oktaf: adalah interval delapan nada, dalam musik diatonis oktav mengidentifikasikan pengulangan nada yang sama hanya dalam tingkatan yang semakin tinggi.
