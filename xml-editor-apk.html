2. carilah apa kepanjangan istilah pada pemrograman Android berikut dan jelaskan fungsinya masing-masing a. JDK b. JREc. SDKd. IDEe. XMLf. APK​


Jawaban:

JDK

ADALAH SEBUAH PERANGKAT LUNAK YG DIGUNAKAN UNTUK MELAKUKAN PROSES KOMPILASI DARI KODE JAVA KE BYTECODE YG DAPAT DIMENGERTI DAN DAPAT DIJALANKAN OLEH JRE ( JAVA RUNTIME ENVIROTMENT )

JRE

( JAVA RUNTIME ENVIROTMENT ADALAH SEBUAH PAKET APLIKASI YG BERISI JVM ( JAVA VIRTUAL MACHINE ) SERTA BEBERAPA LIBRARY KODE PROGRAM TAMBAHAN YG DIPERLUKAN UNTUK MENJALANKAN APLIKASI JAVA.

SDK

SDK ADALAH PENGEMBANGAN APLIKASI PERANGKAT LUNAK.PENGEMBANG APLIKASI MENGGUNAKAN SDK TERTENTU UNTUK FUNGSIONALITAS CANGGIH, PEMBERITAHUAN PUSH, DAN IKLAN KE APLIKASI.

IDE

IDE ( INTEGRATED DEVELOPMENT ENVIRONMENT ) ADALAH PROGRAM KOMPUTER YG MEMILIKI BEBERAPA KUALITAS YG DIGUNAKAN DALAM PEMBANGUNAN PERANGKAT LUNAK.TUJUAN IDE ADALAH UNTUK MENYEDIAKAN SEMUA UTILITAS YG DIPERLUKAN DALAM PEMBANGUNAN PERANGKAT LUNAK.

XML

XML ( EXTENSIBLE MARKUP LANGUAGE ) ADALAH BAHASA MARKUP UNTUK KEPERLUAN UMUM YG DISARANKAN OLEH W3C UNTUK MEMBUAT DOKUMEN MARKUP KEPERLUAN PERTUKARAN DATA ANTAR SISTEM YG BERANEKARAGAM.

APK

APK ( ANDROID PACKAGE KIT ) ADALAH FORMAT FILE YG DIGUNAKAN OLEH SISTEM OPERASI ANDROID UNTUK DISTRIBUSI DAN PEMASANGAN APLIKASI SELULER.
