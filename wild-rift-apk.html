6. great rift valley terdapat dibenua


Great Rift Valley atau dikenal sebagai Lembah Celah Besar adalah dataran rendah berbentuk linear di benua Afrika, yang terletak di antara beberapa dataran tinggi atau pegunungan, dan terbentuk akibat keretakan geologis atau patahan.

Retakan tersebut bahkan membelah jalan raya tersibuk di Kenya, tepatnya di Mai Mahiu-Narok. Kedalaman retakan mencapai 50 kaki, dengan lebar 20 meter. Menurut para ilmuwan, tempat ini hanyalah satu dari ratusan titik rentan di sekitar Great Rift Valley.

Dengan adanya fenomena geologi ini, empat negara yang berada di Tanduk Afrika (Horn of Africa) -- Somalia, setengah dari Ethiopia, Kenya dan Tanzania; diperkirakan akan terpisah dari Afrika untuk membentuk benua baru.

28. change the sentence into positive from.(?) Is snake wild animal ?(+)a snake ia not wild animalb snake ia wild not animalc snake isn't wild animald snake ia wild animal​


Jawaban

Change the sentence into positive form!

( ? ) Is snake wild animal?

( + ) D. Snake is wild animal.

Penjelasan

kalimat di atas menggunakan Simple Present Tense. Penggunaan Simple Present Tense adalah untuk menyatakan fakta (fact), kebiasaan (habit), kebenaran umum (general truth), situasi yang permanen (permanent situation). Rumus kalimat simple present tense, antara lain:

Positive form    

( + ) subject + verb 1 (-s/-es/ies) + object

( + ) subject + to be (is/am/are) + object

I teach mathematic. Rina is a magician. Negative form

( - ) subject + do/does + NOT + verb 1 + object

( - ) subject + to be (is, am, are) + not + object

I do not cook breakfast. My parents are not Indonesians. Interrogative form

( ? ) do/does + subject + verb 1 + object + ?

( ? ) to be (is/am/are) + subject + object + ?

Do I know you? Is she a teacher?

Penambahan akhiran -s/-es/-ies pada simple present tense hanya digunakan pada kalimat positive ( + ) ketika subjek ketiga tunggal, seperti he, she, dan it atau mother, father, friend, teacher (sebutan atau jabatan), (a) cat, (a) goose (hewan), (a) car, (a) house (benda), Dian, Mr. Risa, Joyce (nama orang), dan sebutan subjek ketiga tunggal lainnya.
