8. Apa yang di maksud (2nd)


2nd adalah penulisan ordinal number dalam bentuk angka dari "Second", yang berarti "Kedua".

✦꧁Pembahasan꧂✦

Ordinal Numbers

Ordinal numbers adalah cara penulisan angka dalam Bahasa Inggris yang menunjukkan urutan, tahapan, atau peringkat, misalnya pertama, kedua, ketiga, dan seterusnya. Berbeda dengan angka yang menunjukkan jumlah atau penghitungan matematika, ordinal number memiliki cara penulisannya sendiri.

Cara Penulisan Ordinal Numbers dalam Tulisan

Caranya adalah penambahkan "th" di ujung kata angkanya, namun terdapat pengecualian untuk angka 1, 2, 3, dan angka-angka majemuk yang memiliki akhiran angka 1, 2, atau 3.

One = First (Pertama)Two = Second (Kedua)Three = Third (Ketiga)Four + th = Fourth (Keempat)Five + th = Fifth (Kelima)Six + th = Sixth = (Keenam)Seven + th = Seventh (Ketujuh)Eight + th = Eighth (Kedelapan)Nine + th = Ninth (Kesembilan)Ten + th = Tenth (Kesepuluh)Eleven + th = Eleventh (Kesebelas)Twelve + th = Twelfth (Keduabelas)Thirteen + th = Thirteenth (Ketigabelas)Fourteen + th = Fourteen (Keempatbelas)Fifteen + th = Fifteenth (Kelimabelas)Sixteen + th = Sixteenth (Keenambelas)Seventeen + th = Seventeenth (Ketujuhbelas)Eighteen + th = Eighteenth (Kedelapan belas)Nineteen + th = Nineteenth (Kesembilanbelas)Twenty + th = Twentieth (Keduapuluh)

Adapun untuk angka puluhan, ratusan, maupun ribuan, dan seterusnya, cara penulisan ordinal numbernya adalah dengan menambahkan "th". Namun jika akhirannya adalah 1, maka kata "one" dihapus dan tambahkan "first". Jika akhirannya 2, maka kata "two" dihapus dan tambahkan "second". Jika akhirannya 3, maka "three" dihapus dan tambahkan "third". Contoh :

One hundred forty-six + th = One hundred forty sixth (Keseratus empat puluh enam)Three hundred sixty-one + first = Three hundred sixty first (Ketigaratus enam puluh satu)Twenty-two + second = Twenty second (Keduapuluh dua)Sixty-tree + third = Sixty third (Keenampuluh tiga)

Cara Penulisan Ordinal Numbers dalam Angka

Caranya adalah dengan melihat akhirannya, apakah th, st, nd, atau rd.

First = 1stSecond = 2ndThird = 3rdFourth = 4thFifth = 5thTwenty first = 21stThirty second = 32ndFifty third = 53rdSeventy seventh = 77thOne hundred fifty sixth = 156th
