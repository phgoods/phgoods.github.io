7. apa fungsi dari google chrome?


Fungsi dari Google Chrome adalah utamanya untuk browsing. Google Chrome sendiri juga memiliki fungsi tambahan lain yang memungkinkan penggunanya mensinkronisasi akun Google dengan browser tersebut, pengguna dapat memasang ekstensi yang menunjang kinerja, membuka beberapa bentuk file didalam browser, dan melakukan inspect element pada suatu webpage.

Pembahasan

Google Chrome adalah salah satu produk hasil pengembangan perusahaan Google yang tersedia sebagai web browser maupun di android.  Pengguna Google Chrome sendiri kian meningkat tiap tahunnya dikarenakan fitur yang semakin lengkap dan interface yang lebih user-friendly.

Fungsi utama browser ini adalah untuk menjelajah berbagai web untuk kebutuhan berkirim email, layanan sosial media, search engine dan sebagainya.

Fungsi tambahan browser ini antara lain :

→ Memungkinan pengguna untuk mensinkronisasi akun Google dengan Chrome.

Biasanya ketika pengguna pertama kali menggunakan Chrome akan ditawarkan untuk menghubungkan akun Google aktif pengguna agar dapat terhubung sehingga memudahkan dalam mengintegrasikan fasilitas google yang ada pada android dengan browser. Contohnya, Google Calendar, rekomendasi keyword pada kolom search Google, dan login otomatis pada layanan Google lainnya

→ Pengguna dapat memasang ekstensi yang menunjang kinerja

Ekstensi yang tersedia di Chrome sangat beragam dan akan sangat mendukung kinerja penggunanya. Contoh : pemblokir iklan, ekstensi tools perusahaan, ekstensi mempercepat download/upload file dan sebagainya.

→ Membuka beberapa bentuk file didalam browser

Chrome selain sebagai browser untuk menjelajah web, browser ini dapat digunakan untuk membuka file seperti .pdf, docx, .mp3(audio), mp4 (video)

→ Melakukan inspect element pada suatu webpage

Hal ini dapat dilakukan di Chrome dengan tujuan mengedit suatu tampilan web pada susunan HTML / CSS namun hal tersebut hanya dapat dilihat pada komputer pengguna yang melakukan perubahan denga inspect element.

Jawaban:

Berikut ini Fitur Dan Fungsinya :

1. Meminimalkan Tab

Jujurlah, kita semua terkadang memiliki lebih dari selusin tab dibuka di komputer kita. Salah satu trik rapi untuk tetap membuat sebagian besar dari mereka tetap terbuka adalah dengan mengubahnya menjadi tab yang disematkan. Di Google Chrome Anda dapat mengklik kanan tab dan memilih opsi “Pin Tab” atau “Pasang Pin”. Ini mengubah tab menjadi ikon yang memungkinkan Anda untuk melanjutkan multitasking.

Manfaat besar dari fitur “Pin Tab” adalah Anda tidak dapat menutup tab ini secara tidak sengaja karena tanda “X” (tutup /close) menghilang setelah menyematkannya.

2. Mode penyamaran (Incognito)

Google Chrome sangat mudah digunakan dan intuitif. Namun, Google mengumpulkan data penelusuran kita. Jadi untuk memperbaiki ini, Anda dapat menggunakan Mode Penyamaran. Fitur ini tidak menyimpan riwayat penelusuran atau unduhan Anda. Anda dapat mengaktifkan atau mengaksesnya dengan tiga cara berbeda:

Tekan Ctrl / Command + shift + N

Pilih Menu File dan pilih Jendela Penyamaran Baru

Unduh ekstensi Jendela Penyamaran Baru

Fitur ini sangat berguna jika Anda lebih suka menyimpan riwayat penelusuran dan memanfaatkannya untuk iklan di masa mendatang atau halaman yang disarankan.

3. Simpan Halaman Web sebagai File PDF

Pernahkah Anda melihat-lihat informasi yang menarik atau penting dan kemudian lupa untuk membookmark atau menyimpannya di “favorit”, sehingga tidak mungkin untuk menemukan lagi? Kemungkinan Anda sering melakukan ini bukan?

Untungnya, ada solusi yang mudah. Anda dapat menyimpan halaman web sebagai file PDF. Pada keyboard Anda, tekan control / command + p dan Anda akan dapat menyimpan halaman web sebagai PDF.

4. Buka Tab yang baru saja ditutup

Pernahkah puluhan tab dibuka dan tiba-tiba Anda tanpa sengaja menutup browser Anda? Itu mungkin terjadi pada kita semua. Anda dapat dengan mudah memulihkan semua tab menggunakan dua pendekatan. Jangan panik jika ini terjadi karena ada solusi dan solusi untuk itu.

Salah satunya adalah dengan menekan Ctrl / CMD + Shift + T.

Cara lainnya adalah dengan mengklik pada tiga titik vertikal pada browser Anda dan arahkan kursor ke “History”.

5. Memecahkan Masalah Matematika

Peramban Google Chrome tidak hanya mencari informasi yang relevan dan diperbarui. Ia juga mampu melakukan beberapa masalah matematika. Di dalam omnibox (bilah alamat atau URL Chrome), Anda dapat melakukan latihan matematika.

Misalnya, jika Anda berjuang dengan persentase, Anda dapat mencari 20 persen dari jumlah x dan itu akan langsung memberikan hasil. Cukup praktis, bukan ?!

6. Memutar File Media

Apakah Anda sering menemui kesulitan saat memutar atau menonton file video? Nah, sekali lagi Chrome datang untuk menyelamatkan. Anda dapat mendengarkan atau memutar video dari semua jenis file film atau musik (mp3, mp4, .mov, .mkv, .ogv, .webm, .wav, dll.) Hanya dengan menyeret file ke bilah pencarian.

Selain itu, Anda juga dapat melihat gambar, file PDF, dan file Microsoft Office.

7. Menavigasi Dengan Cepat Antara Tab

Dengan banyaknya tab yang Anda buka, tentunya Anda akan kesulitan melakukan navigasi. Daripada mengklik setiap tab, Anda dapat menggunakan tombol pintas seperti Ctrl + Tab untuk menavigasi semua tab yang berbeda. Anda juga dapat menavigasi ke tab pertama dengan menekan Ctrl-1, Ctrl-2, dan seterusnya. Jika Anda ingin beralih ke tab terakhir, tekan Ctrl-9.

8. Stay Focusd

Internet saat ini memiliki kemampuan luar biasa. Terkadang kita ingin menyelesaikan pekerjaan, tetapi kita semua adalah manusia. Kita terkadang menunda-nunda dengan mengunjungi situs web yang sangat kami sukai, atau mungkin beristirahat sejenak dengan menonton film di Netflix, video di YouTube, atau menjelajahi Facebook.

Dengan ekstensi StayFocusd Chrome, Anda dapat benar-benar tetap fokus dan menyelesaikan lebih banyak dalam waktu lebih singkat.

Extensions ini secara alami membantu Anda tetap lebih produktif dengan membatasi jumlah waktu yang Anda habiskan di situs web. Anda dapat mengatur waktu dan secara otomatis akan memblokir situs-situs tersebut setelah periode tertentu. Untuk menggunakannya, Anda dapat mengunjungi tautan ini.

9. Grammarly untuk Editing

Grammarly harus dimiliki terutama bagi Anda yang terbiasa bekerja dengan mengetik banyak teks di chrome seperti pembuatan konten, digital marketing, copy writer, atau sebagainya. Grammarly membantu Anda memeriksa tata bahasa dan ejaan untuk semua yang Anda tulis online.

Anda dapat menggunakannya secara profesional atau sebagai pemula, yang akan membuat proses pengeditan jauh lebih mudah dan lebih efisien. Selain itu, secara otomatis dapat memeriksa kesalahan ketik ketika Anda mengirim email, mengetik Tweet, atau membuat konten penting yang berhubungan dengan bisnis.Tentu Anda tidak mau bisnis Anda terlihat buruk karena banyak typo pada konten atau tulisan yang Anda buat bukan?
