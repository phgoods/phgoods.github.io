› Pendahuluan

Pertidaksamaan Linear Satu Variabel adalah pertidaksamaan linear yang memuat satu variabel dengan pangkat tertinggi satu dan memuat tanda pertidaksamaan (<, >, ≤, ≥).

Bentuk Umum :

ax + b < c atau ax - b < 0

ax + b > c ata ax + b > 0

__________________________

ax + b ≤ c atau ax + b ≤ 0

ax + b ≥ c atau ax + b ≥ 0

__________________________

Keterangan :

x = Variabela = Koefisien xb dan c = Konstanta

› Pembahasan

2(x - 5) > 20

→ 2x - 10 > 20

→ 2x > 20 + 10

→ 2x > 30

→ x > 30 ÷ 2

→ x > 15

› Kesimpulan

x > 15
