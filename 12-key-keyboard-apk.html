21. Sebutkan 5 tombol keyboard shortcut key yang dapat digunakan pada ms excel,jelaskan​


Jawaban:

Shortcut :(Gabungan Tombol Dasar)

1.)Ctrl + C berfungsi untuk melakukan copy pada obyek yang dipilih (memilih obyek, fungsinya hampir sama seperti Ctrl + X namun shortcut ini berfungsi untuk memilih).

2.)Ctrl + S berfungsi untuk melakukan save pada program yang sedang diproses, biasanya berlaku pada microsoft word, microsoft powerpoint, notepad, paint, ataupun microsoft excel.

3.)Ctrl + A berfungsi untuk memilih atau select seluruh obyek pada program yang sedang diproses.

4.)Ctrl + X berfungsi untuk melakukan cut pada obyek yang dipilih (memotong obyek, fungsinya hampir sama seperti Ctrl + C namun shortcut ini berfungsi untuk memotong).

5.)Ctrl + V berfungsi untuk melakukan paste obyek yang telah di cut ataupun copy.

Tombol Dasar :

1.)Esc atau Escape adalah tombol yang biasanya digunakan untuk menuju main menu atau menghentikan program yang diproses (seperti powerpoint, security option yaitu Ctrl + Alt + Delete).

2.)Backspace adalah tombol yang digunakan untuk menghapus kata - kata yang ditulis ataupun space pada kalimat dan kata - kata.

3.)Enter adalah tombol yang digunakan untuk memulai proses.

4.)Capslock adalah tombol yang digunakan untuk mengaktifkan keyboard dalam mode huruf kapital.

5.)Print Screen adalah tombol yang digunakan untuk melakukan tangkapan gambar layar atau screenshoot pada suatu monitor sesuai dengan obyek dan program yang sedang dijalankan, jadi tidak mungkin menangkap desktop kosong ketika masih menjalankan obyek dan program (apabila tidak diminimize).

26. Jelaskan dan sebutkan fungsi dari tombol-tombol keyboard: a.numeric key b.alphabet key c.function key d.special function key


Jelaskan dan sebutkan fungsi dari tombol-tombol keyboard:a.numeric keyb.alphabet keyc.function keyd.special function keyJawaban Pendahuluan

Soal diatas adalah soal mengenai beberapa tombol pada keyboard, seperti kita ketahui bahwa keyboard merupakan salah satu perangkat input pada komputer. Beberapa tombol pada keyboard memiliki fungsi khusus.

Pembahasan

Beberapa tombol khusus pada keyboard yang akan kita bahas adalah sebagai berikut

a. Numeric key, numeric key pada keyboard berfungsi untuk memberikan inputan berupa angka dan operator matematika seperti penjumlahan dan pengurangan dan masih banyak lainnya.

b. Alphabet key, berfungsi untuk memberikan inputan berupa huruf dan tanda baca. Inputan angka juga termasuk ke dalam alphabet key.

c. Function key, berfungsi untuk mengaktifkan menu tertentu pada suatu aplikasi ketika tombol tersebut ditekan. Function keyboard pada keyboard adalah tombol F1 hingga F12 yang biasanya terdapat pada bagian atas keyboard.

d. Special function key, tombol ini bukan merupakan tombol untuk memberikan inputan. Tombol special function key hampir sama dengan function key, namun tombol ini berfungsi ketika ditekan bersamaan dengan tombol lain. Tombol special funtion key antara lain adalah CTRL, ATL. Misal untuk memunculkan task manager adalah dengan menekan CTRL+ALT+DEL.

Kesimpulan

Demikian pembahasan mengenai beberapa tombol pada keyboard, dengan memahami tombol-tombol pada keyboard maka dapat meningkatkan kecepatan kita dalam menyelesaikan suatu pekerjaan atau tugas.
