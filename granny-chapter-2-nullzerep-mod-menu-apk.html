Pembahasan

Turbo Pascal adalah sebuah sistem pengembangan perangkat lunak yang terdiri atas compiler dan lingkungan pengembangan terintegrasi atas bahasa pemrograman pascal untuk sistem operasi SP/M, CP/M-86 dan MS-DOS.

Program adalah instruksi-instruksi yang diberikan kepada komputer untuk melaksanakan tugas tertentu.

Pemrograman adalah upaya untuk membuat kode-kode yang dapat dikenal oleh komputer.

Turbo Pascal yang dikeluarkan oleh Borland International dirilis pada tahun 1992 yang mempunyai :

Compiler : Untuk mengkompilasi program menjadi suatu objek yang dipahami oleh komputer

Debugger : Untuk membantu mencari kesalahan program

Linker : Untuk menggabungkan file objek dan file pustaka dan membentuk file executable

Editor : Untuk menulis program sumber /source code

Turbo Pascal mempunyai sebaris menu dibagian paling atas yang terdiri dari :

1. File

New : Membuat program baru

Save As : Menyimpan program dengan nama yang lain

Save All : Menyimpan semua program yang ada di layar

Change Dir : Mengubah drive aktif untuk penyimpanan dan pemanggilan file

Print : Untuk mencetak

DOS Shell : Untuk masuk ke DOS sementara dan kembali lagi ke Turbo Pascal ketik EXIT kemudian Enter

Exit : Keluar dari Turbo Pascal

Undo : Untuk mengembalikan yang terhapus terakhir

Redo : Untuk menghapus kembali yang di Undo

Cut : Untuk menghapus blok dan dipindah ketempat lain

Copy : Untuk mengcopy blok

Paste : Untuk menampilkan isi dari cut atau copy blog

Show Clipboard : Untuk menampilkan isi clipboard

3. Search

Find : Untuk mencari atau perintah yang tertulis pada program yang sedang dikerjakan

Replace : Untuk mengganti kata atau perintah yang tertulis pada program yang sedang dikerjakan

Search Again : Untuk mencari lagi kata atau perintah yang tertulis pada program yang sedang dikerjakan

Go to line number : Untuk menuju kebaris yang dimaksud

Show last compiler error : Untuk menunjukkan kesalahan kompilasi yang terakhir

Find eror : Untuk mencari kesalahan

Find procedure : Untuk mencari procedure yang dimaksud

Previous Browserdan Globals : Untuk menunjukkan berbagai fungsi dan prosedur yang ada di pascal

Run : Untuk menjalankan atau mengekskusi program

Step Over

Trace into : Untuk mengeksekusi baris berikutnya

Go to cursor : Untuk memulai/melanjutkan eksekusi program dimulai dari posisi saat ekskusi dimulai sampai tempat kursor berada.

Program reset : Untuk memberitahukan Turbo Pascal bahwa sudah selesai dengan pembetulan-pembetulan dan menginisialisasi debugger untuk operasi yang lain

Make : Untuk mengaktifkan perintah make, jika file yang primer telah diberi nama, maka file ini akan dikompilasi, jika tidak maka file yang terakhir yang sedang dimuat di editor akan dikompilasi.

Add breakpoint : Untuk menambah breakpoint

Go to Next : Untuk menuju ke baris selanjutnya

Go to previous : Untuk menuju ke baris sebelumnya

Grep : Untuk menyaring masukan atau menampilkan baris-baris yang mengandung kriteria tertentu

8. Option : Untuk mengatur berbagai pilihan computer, memori size, linker, debugger

9. Window : Untuk mengatur tampilan windows

10. Help : berisi berbagai bantuan yang disediakan Turbo Pascal
