5. Cara Mengupradge Android Jelly Bean Ke Nougat?


mungkin mengupgrade dari Android Jelly Bean ke Nougat tak semudah itu karena perlu waktu yang lama. jadi untuk mengupgradenya perlu:
1. koneksi internet stabil dan memadai
2. dukungan Android dari pembuat ponsel
3. dukungan upgrade lewat udara
4. WI-FI yang stabil koneksinya
5. baterai minimal 50% keatas
6. data wajib dibackup

jadi disini akan menjelaskan tentang upgrade melalui udara:
1. ke setelan => tekan pembaruan nirkabel => tekan check for updates
2. jika ada update terbaru, file akan terunduh. dan jika file sudah terunduh, kita akan diminta untuk menghidupkan ulang hp
3. ketika berada didalam update, tunggu hingga selesai dan jangan matikan ponsel. jika terjadi, ponsel akan mengalami kerusakan software (bootloop/brick)

kalau tidak bisa, dapat menggunakan file yang diunduh dari internet
1. jika updatenya pakai hp (langsung), pergi ke pilihan yang sama
2. kelocal updates lalu pilih file
3. jika ada peringatan seperti digambar kedua, pastikan bahwa file sudah benar dan lakukan update sekarang
4. tunggu hingga update selesai

_________________________________________________________
kalau dari Kitkat ke lollipop, menurut saya bisa. itu juga tergantung kepada manufaktur ponselnya apakah ponselnya masih didukung apa tidak. untuk mengupgrade versi androidnya, kamu perlu mengecek pembaruan dari "check for updates" di pengaturan. setelah ada pemberitahuan bahwa ada firmware terbaru, download firmwarenya. tetapi, ada hal-hal yang harus diperhatikan yaitu:
a. sisa penyimpanan telepon apa masih ada apa enggak (minimal 2gb atau lebih)
b. persentase baterai ponsel (lebih dari 50%)
c. koneksi internet. untuk unduh firmware (koneksi harus stabil. disarankan untuk menggunakan WI-FI)
ketika semua sudah disediakan, lakukan update ketika bilah notifikasi sudah menampilkan notifikasi bahwa ada update baru. ketika kamu mengkliknya, hp akan reboot dan masuk kedalam proses update. setelah instal update, biasanya akan reboot berulang kali. ini menandakan bahwa update masih berlanjut

Q: upgrade kok pakai internet
A: karena file untuk upgradenya akan diunduh dan dipasang langsung ke ponselmu

Q: apa yang terjadi jika updatenya gagal?
A: hpnya kena masalah firmware (bootloop, dll)

Q: adakah cara lain selain dengan cara ini?
A: ada kok. tetapi, kamu harus menanggung resikonya sendiri yaitu menginstal update perangkat dengan file diinternet
