30. Google drive adalah layanan google untuk.....


Google drive merupakan salah satu layanan google yang berfungsi untuk berbagai keperluan seperti:

Digunakan sebagai tempta penyimpanan file baik itu file yang dimasukkan langsung atau file back up seperti file back up dari aplikasi WhatsApp.Digunakan sebagai saranan sharing file antara sesama. Digunakan sebagai tempat penyimpanan link browser yang telah dikunjungi sehingga memudahkan untuk dikunjungi kembali pada masa yang akan datang. Digunakan sebagai tempat pengeditan beberapa jenis file seperti file dalam bentuk excel dan power point. Pembahasan

Google drive merupakan salah satu layanan google yang sering digunakan untuk berbagai kepereluan yang berhubungan dengan menyimpan file, memback up file dan mengedit file. Google drive memiliki dua layanan berbeda yaitu layanan versi gratis dan layanan versi berbayar. Kapasitas penyimpanan layanan versi gratis sudah pasti lebih sedikit jika dibandingkan dengan layanan versi berbayar. Di mana layanan versi gratis memberikan akses penyimpanan  maksimal hingga 15 GB dan layanan versi berbayar memberikan akses penyimpanan hingga 100 GB, 200 GB hingga 2 TB.
