=>>soal:

Tentukan fungsi kuadrat yg memotong sumbu x di titik ( 6, 0 ) dan ( 12, 0 ) serta memotong sumbu y di titik ( 0, - 20 )

=>>penyelesaian soal:

Grafik fungsi kuadrat memotong sumbu x di titik ( 6, 0 ) dan ( 12, 0 ) maka y = a ( X - 6 ) ( X - 12 )

karena Grafik melalui titik ( 0, - 20 ) maka berlaku

- 20 = a ( 0 - 6 ) ( 0 - 12 )

= 72 a

[tex] \: \: \: a \: = \: \frac{ - \: 5}{18} [/tex]

dgn demikian, fungsi kuadrat itu adalah:

[tex]y \: = \: ( \: \frac{ - \: 5}{18} \: ) \: ( \: x \: - \: 6 \: ) \: ( \: x \: - \: 12 \: ) \\ \\ \: = \: \frac{ - \: 5}{18} \: ( \: {x}^{2} \: - \: 12x \: - \: 6x \: - \: 72 \: ) \\ \\ = \: \frac{ \: - \: 5}{18} \: ( \: {x}^{2} \: - \: 18x - \: 72 \: ) \\ \\ = \: \frac{ - \: 5}{18} \: {x}^{2} \: + \: \frac{90}{18}x \: - \: \frac{360}{18} \\ \\ = \: \frac{ - \: 5}{18} {x}^{2} \: + \: 5x \: - \: 20[/tex]

=>>kesimpulannya:

jadi, fungsi kuadrat yg memotong sumbu x di titik ( 6, 0 ) dan ( 12, 0 ) adalah - 5/18 + 5x - 20
