4. Apakah fungsi dari MAC Access Filtering ,MAC Filtering, MAC APBlockir , NIC Filtering ,dan MAC Security ?


[jawaban]

Mata Pelajaran: Teknologi Informasi

• Jawaban pendek:

Apakah fungsi dari MAC Address Filtering ,MAC Filtering, MAC APBlockir , NIC Filtering ,dan MAC Security ?
1. MAC Address Filter berfungsi untuk membantu mencegah pengguna asing (tidak diinginkan) yang berniat untuk mengakses masuk ke jaringan router nirkabel.

2. MAC Filtering berfungsi untuk melakukan penyaringan (filter) pada MAC address tertentu dan membuat aturan apakah suatu MAC address dapat mengakses jaringan wireless atau tidak.

3. MAC APBlockir berfungsi untuk memblacklist / memblokir AP Tertentu

4. MAC Security berfungsi untuk melindungi PC dari serangan virus atau malware dari luar.


• Jawaban panjang: 

MAC Address Filter
MAC Address Filter ini berfungsi untuk membantu anda untuk mencegah pengguna asing (tidak diinginkan) yang berniat untuk mengakses masuk ke jaringan router nirkabel anda. Dengan menerapkan fitur ini, maka hanya perangkat nirkabel yang memiliki alamat MAC yang telah terdaftar (ditetapkan) saja yang dapat memperoleh akses ke router nirkabel

MAC Filtering
MAC filtering adalah suatu utility pada access point untuk melakukan filter pada MAC address tertentu dan membuat aturan apakah suatu MAC address dapat mengakses jaringan wireless atau tidak.
