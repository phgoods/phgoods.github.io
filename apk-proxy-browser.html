Penjelasan:

Proxy server adalah server yang bertindak sebagai perantara antara client dan server tujuan. Saat client mengakses sebuah situs web, alamat IP client akan terlihat oleh server tujuan. Namun, jika client menggunakan proxy server, maka alamat IP yang terlihat oleh server tujuan adalah alamat IP proxy server, bukan alamat IP client.

Transparent proxy serverAnonymous proxyDistorting proxy:High anonymity proxyFitur filtering server
