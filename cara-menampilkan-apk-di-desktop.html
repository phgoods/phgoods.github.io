a. Wallpaper

Wallpaper adalah gambar menempel di atas desktop. Windows menyediakan berbagai macam pilihan walpaper sehingga kita dapat dengan mudah mengubahnya sesuai dengan keinginan .

b. Ikon shortcut

shortcut adalah gambar kecil yang mewakili file,objek,folder,program aplikasi,dan drive pada komputer. ikon shortcut ini untuk memudahkan dalam mengaktifkan suatu program aplikasi. Ikon ini bersifat semipermanen sehingga jumlahnya dapat dikurangi maupun di tambah . Namun,secara default ikon shortcut dalam desktop sebagai berikut:

1) My computer, ikon ini berfungsi untuk mengelola file dan folder pada windows.

2)My document , merupakan suatu folder standart pada windows di mana semua data secara otomatis akan tersimpan pada folder ini

3)internet explorer ,ikon ini berfungsi untuk mengaktifkan software internet explorer. internet explorer adalah ikon yang digunakan untuk menjelajahi dunia maya.

4)Recycle Bin,kegunaan ikon ini adalah untuk menampung sementara file yang kita hapus.

5)Beberapa ikon program aplikasi yang di sesuaikan dengan kegunaan masing masing.

c. Taskbar

Taskbar adalah sarana yang digunakan untuk membuka program serta menjelajahi komputer.Taskbar terletak pada bagian bawah layar windows baik windows XP ataupun Windows 98.

Storm Worm adalah virus jenis Trojan Horse yang dapat menginfeksi PC, kemudian memungkinkan perangkat dikendalikan dari jarak jauh untuk menyebarkan email spam. Virus komputer ini juga berbahaya karena dapat install rootkit untuk menyembunyikan program jahat di PC yang tidak dapat dideteksi antivirus.

ILOVEYOU

Sama seperti Storm Worm, virus ILOVEYOU juga menyebar melalui pesan email bersubjek “I LOVE YOU” di tahun 2000. Ketika virus sudah masuk ke dalam PC, maka ILOVEYOU dapat mengubah file, mengubah malware lainnya, dan mengganti data. Virus komputer ini juga berisikan program VBScript, yang ketika di-klik membuat virus dapat mengakses daftar kontak Outlook korban dan mengirim email sendiri berisikan virus tersebut ke semua orang di dalam kontak.

MyDoom

Pertama kali dideteksi pada 2004, virus komputer ini diklaim menjadi virus dengan penyebaran tercepat yang pernah ada. MyDoom dapat memperlambat akses internet dan menyebabkan jutaan email terinfeksi dalam waktu 36 jam pertama. Tak hanya itu, virus komputer ini juga menyebabkan Microsoft Windows mengalami kerugian finansial hingga USD 38,5 miliar.

Black Shades

Black Shades adalah virus tipe ransomware yang mengenkripsi (mengunci) file, sehingga memungkinkan para peretas mengendalikan PC kamu dari jarak jauh. Begitu masuk, mereka akan memata-matai melalui webcam PC, mencuri file dan informasi akun, dan melihat apa yang kamu ketik. Jadi, untuk memulihkan PC, korban harus membayar sejumlah uang tembusan.
