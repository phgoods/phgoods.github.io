17. Maximum network connection on windows 10 through lan from other computer


Jawaban:

LANKatakanlah Anda memiliki jaringan lebih dari sepuluh komputer Windows yang terhubung dalam workgroup lokal tanpa pengontrol domain. Salah satu komputer memiliki printer yang terhubung, dan lebih dari sepuluh mesin dalam kelompok kerja perlu mencetak pada waktu yang sama.

Nah, Anda mungkin berkata, “Bung, dapatkan server!” atau "Bung, dapatkan server cetak!"

Benar, itu akan menyelesaikan masalah… Tapi terkadang, hal-hal seperti uang dan pengetahuan teknis menjadi faktor pembatas dalam kasus seperti itu.

Kabar baiknya: Sebenarnya sangat mudah untuk meningkatkan batas dan membuat semua orang mencetak/berbagi file dalam waktu yang sangat singkat!

Baiklah, pertama beberapa kata lagi tentang Jaringan Windows. Jika komputer menjalankan Windows XP Home, jumlah maksimum koneksi yang dapat dibuat ke/dari komputer tersebut di LAN adalah 5. Jika mesin menjalankan XP Pro, Vista, dll. maka maksimum adalah 10.

Juga, Anda harus memahami bahwa batas tersebut mengacu pada koneksi, bukan jumlah komputer. Secara teknis, Anda dapat memiliki LAN dengan 500 mesin Windows, dan selama tidak lebih dari 10 komputer mencoba terhubung ke komputer lain, semuanya akan berfungsi dengan baik. Tetapi Anda mungkin masih akan mengalami beberapa sakit kepala seperti ketika Anda menelusuri Network Neighborhood, tidak semua komputer akan muncul. Itu cukup mengganggu.

Jadi sekarang, inilah cara Anda memperbaikinya:

Pertama, ketik Win-Runtuk membuka kotak dialog "Jalankan", dan ketik gpedit.msc.

Pada titik ini, salah satu dari dua hal akan terjadi:

Anda memiliki versi Windows yang lebih bagus, jadi konsol manajemen yang bagus akan muncul.

Anda memiliki versi Windows yang kurang mewah, jadi ini akan memberi Anda pesan kesalahan.

Jika Anda melihat jendela manajemen, navigasikan ini:

Computer Config -> Windows Settings -> Security Settings -> Local Policies -> Security Option

Dan atur Interactive Logonke 50(maksimum) atau 0(yaitu nonaktifkan caching sepenuhnya)

Jika Anda mendapatkan pesan kesalahan alih-alih jendela manajemen, lakukan hal berikut:

Jenis Win-R

Ketik regedt32dan tekan Enter

Cari entri berikut:

HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\CurrentVersion\Winlogon\CachedLogonsCount

Ubah CachedLogonsCountke 50(maks) atau 0(dinonaktifkan)

Tutup Editor Registri

Ulangi pada setiap mesin di jaringan hanya untuk ukuran yang baik, et voila! Kamu sudah selesai.

Saya telah menggunakan trik ini sebelumnya di jaringan Windows multi-versi, dan itu bekerja dengan sangat baik. Ini juga jauh lebih mudah daripada menyiapkan server, dan lebih murah daripada membeli semacam server cetak mandiri.

Terakhir, perhatikan bahwa untuk versi Windows tanpa gpedit.msc, Anda dapat mengunduh file Excel dengan entri registri yang sesuai untuk diedit di Vista (entri biasanya sama di versi Windows lainnya).
