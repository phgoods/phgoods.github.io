18. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.

26. Apakah di boleh kita me reff apk secara terus menerus dan itu pakai apk clone Tolong yg ahli agama dan teknologi bisa dijelaskan (manfaat,mudhorot nya)


Reef APK menggunakan APK Clone ADALAH jelas hal yang terlarang apabila dilihat dari ketentuan Pusat Kebijakan Developer Google Play. Membuat APK dengan cara menggandakan berkas dari yang kemudian dimodifikasi sekedarnya untuk mengelabui pihak Google Play adalah kecurangan. Dari sudut pandang agama, APK clone ini sama saja dengan pencurian.

Pembahasan

APK adalah kependekan dari Application Package File yang dalam bahasa Indonesia disebut dengan Berkas Paket Aplikasi. APK ini adalah format berkas yang dipergukan dalam menyebarkan dan memasan aplikasi pada perangkat smartphone pada berbagai sistem operasi.

APK Clone sendiri adalah program atau aplikasi yang khusus dibuat untuk memudahkan orang-orang yang hendak membuat salinan ekstra dari aplikasi yang sudah ada. Sayangnya, semakin ke sini, semakin banyak yang menggunakan APK Clone ini sebagai jalan pintas untuk membuat aplikasi yang sama sekali tidak baru namun hanya diedit sedemikain rupa sehingga tampilan nampak berbeda dengan aplikasi yang dikloning.

Google Play memiliki kebijakan melarang hal tersebut, selain itu banyak aplikasi juga dilindungi oleh hak cipta sehingga mengkloning aplikasi dan memasangnya di Google Play jelas perbuatan yang terlarang karena sama dengan pencurian sehingga UANG YANG DIHASILKAN DARI APLIKASI tersebut hukumnya HARAM apabila aplikasi tersebut dikloning dengan tujuan mendapatkan penghasilan di Google Play atau tempat lainnya.
