Dalam halaman Peserta Ujian ada judul mata soal yang akan diujikan, klik mata ujian yang telah ditentukan oleh petugas CBT / Pengawas Ujian. Lalu Klik lagi Judul mata soal lalu klik OK/Yes

Setelah itu masuk kehalaman konfirmasi dan ada tombol “ Attemp Quiz Now” setelah anda mengeklik tombol tersebut maka soal akan segera muncul dan waktu ujian mulai berjalan (Pojok Kiri Atas Soal)

Tombol “SAVE WITHOUT SUBMIT” digunakan untuk menyimpan sementara ujian anda.

Tombol “SUBMIT ALL AND FINISH” lalu konfirmasi OK, digunakan apabila anda sudah yakin mengerjakan semua soal dan ingin mengakhiri sesi ujian ini. Saat tombol ini di tekan maka ujian akan berakhir dan data akan dikirim ke server, dan anda tidak dapat masuk kembali ke soal itu, anda dianggap sudah mengikuti dan menyelesaikan ujian.

Untuk berpindah ke halaman soal berikutnya atau sebelumnya, cukup menekan tombol NEXT / PREV atau Angka halaman di bawah soal ujian.

Jawaban soal dapat diubah kembali sebelum anda menekan SUBMIT ALL and FINISH atau sebelum waktu habis.

Saat waktu habis soal akan otomatis tertutup dan mengirim jawaban sesuai apa yang anda isi sebelum waktu habis.

Setelah selesai ujian atau waktu habis anda harus keluar dari aplikasi CBT dengan klik tombol “LOGOUT” terlebih dahulu.

Tinggalkan bilik komputer, periksa barang anda jangan sampai tertinggal, posisikan kelengkapan komputer seperti semula dan perhatikan Safety Procedure dan Tata Tertib di Ruang CBT.

29. apa syarat syarat laptop untuk CBT?


Spesifikasi Komputer Server UNBK /CBT

Dalam pelaksanaan UNBK diperlukan adanya komputer server yang digunakan untuk menampung data ujian sementara. Spesifikasi komputer server CBT / UNBK yang di minta oleh Kemdikbut minimal sebagai berikut.

Komputer Desktop ( Bukan Laptop ) Memiliki Processor minimal 4 Core dengan clock rate 1.6 GHz ( 64 bit ) Memori RAM minimal 8 GB DDR 3 Kapasitas Harddisk minimal 250 GB Sistem Operasi : Windows Server / Windows 8 / Windows 7 / Linux Ubuntu 14.04 ( 64 bit ) Software Aplikasi : VM Virtual Box, Google Chrome, Exam Browser Admin Memiliki LAN CARD (NIC) 2 unit yang support GigaByte Jumlah Komputer Server minimal 2 Unit, 1 Unit digunakan untuk cadangan. Dilengkapi dengan UPS ( tahan 15 menit )

Spesifikasi Komputer Client UNBK /CBT

Komputer Client digunakan peserta ujian untuk mengakses aplikasi UNBK yang berbasis web, sehingga spesifikasi komputer Client CBT / UNBK tidak perlu tinggi. hanya saja jumlah dari komputer client CBT harus mengikuti rasio 1:3 (1 client untuk 3 peserta) dan jumlah komputer Client yang dihubungkan dalam 1 server tidak boleh lebih dari 40 Unit. Kemdikbut merekomendasikan spesifikasi komputer Client UNBK minimal sebagai berikut.

Komputer Desktop atau Laptop Memiliki monitor minimal 11 inch Menggunakan Processor minimal single core Memori RAM minimal 512 MB Sistem Operasi : Windows XP/Windows 7/Windows 8/ LINUX / MAC / Chrome OS Software Aplikasi : Web Browser ( Chrome / Mozilla Firefox ) dan Xambro Kapasitas Hardisk minimal tersedia 10 GB (free space) Memiliki LAN Card Memiliki sound card lengkap dengan Headset / earphone ( untuk ujian listening SMA/MA dan SMK ) Memiliki cadangan minimal 10% dari total komputer Client.

Selain sekolah perlu mempersiapkan perangkat komputer, untuk dapat melaksanakan kegiatan UNBK ini sekolah juga perlu menyiapkan perlengkapan tambahan yaitu :

Jaringan LAN : Switch 10/100/1000 Mbps , Satu Switch untuk 1 server , Kabel Jaringan + Konektor Jaringan Internet dengan bandwith minimal 1 MBpsPembahasan

Sistem penyelenggaraan ujian secara umum dibagi menjadi 2: Tes Konvensional dan Tes Modern. Tes Modern ini nanti dibagi lagi menjadi linear CBT dan CAT. Apa sih bedanya?

Tes Konvensional: tes dengan menggunakan kertas dan pensil, dengan jumlah butir soal tertentu (fixed length), penilaian secara manual oleh tenaga manusia. Seluruh pelajar Indonesia pastinya udah familiar banget dengan cara ini. Tes Modern: tes dengan menggunakan komputer (computer based test) CAT: tes dengan menggunakan komputer, di mana program mampu memilihkan soal-soal yang tingkat kesulitannya sesuai dengan kemampuan peserta ujian. Sistem tes CAT biasanya dipake untuk simulasi tes CPNS. Linear CBT: tes dengan menggunakan komputer, dengan jumlah butir soal tertentu (fixed length), penilaian dilakukan secara otomatis oleh komputer.

Secara teknis pelaksanaan UNBK saat ini menggunakan sistem semi online, dimana data peserta dan soal akan dikirim dari server pusat ke server lokal melalui jaringan internet ( Sinkronisasi ). Kemudian server lokal akan melayani permintaan data dari komputer Client tanpa menggunakan akses internet.

Kelancaran pelaksanaan UNBK tidak lepas dari spesifikasi komputer CBT / UNBK, bahkan demi menjaga kelancaran pelaksanaan UNBK pihak Kemdikbut memberikan persyaratan wajib bagi Sekolah yang ingin melaksanakan kegiatan UNBK.
