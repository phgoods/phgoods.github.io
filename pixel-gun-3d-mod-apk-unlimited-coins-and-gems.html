Penjelasan dengan langkah-langkah:

Number of $5 coins = x

Number of $2 coins = y

Total value is $38:

5x + 2y = 38 ... (1)

If two $5 coins used, then the number of $5 and $2 coins will be the same:

y = x - 2 ... (2)

Substitute the value of y to equation (1):

5x + 2y = 38

5x + 2(x-2) = 38

5x + 2x - 4 = 38

7x - 4 = 38

7x = 38 + 4

7x = 42

x = 42/7

x = 6

y = x-2

y = 6-2

y = 4

Therefore:

The original number of $5 coins is 6.The original number of $2 coins is 4.
