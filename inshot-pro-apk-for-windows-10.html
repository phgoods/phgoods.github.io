17. Apakah Perbedaan Dari Edisi Windows 10. Yaitu Home, Pro, Enterprise Dan Education​


Jawaban:Versi Windows 10 Home ini dirancang oleh pihak Microsoft untuk memenuhi  kebutuhan ringan harian di rumah saja. Meskipun tidak selengkap seperti Windows 10 versi Pro, Enterprise, dan Education, setidaknya versi ini memang sudah lebih dari kata cukup untuk dapat menangani beberapa aktivitas ringan. Hal ini dikarenakan fitur unggulan yang diberikan oleh Microsoft sudah benar-benar tertanam pada sistem operasi versi ini. Imbas yang terdampak dari adanya pengurangan fitur pada versi Windows 10 Home. Jadi, sistem versi ini yang harganya paling murah. Yaitu sekitar 1,5 jutaan.

Versi Windows 10 Pro ini dapat dibilang cukup popular di berbagai kalangan. Hal ini disebabkan karena fitur yang mereka tawarkan memang tidak lebih serta tidak ada yang kurang. Jika kamu seorang pelajar, mahasiswa, ataupun seorang pebisnis memang rasanya cocok untuk menggunakan Pro. Harga yang dipatok mencapai 2,6 jutaan.

Versi Windows 10 Enterprise ini jarang ditemui di wilayah Indonesia. Bukan tanpa sebab, pihak Microsoft baru saja melepas versi Home dan Pro mereka saja ke pasar tanah air kita ini. Jadi, untuk saat ini jika ingin mendapatkan Windows 10 Enterprise hanya dapat membelinya dari pasar luar negeri. Harganya dibandrol sekitar 1,4 jutaan.

Windows 10 Education ini sama seperti versi Enterprise, yang juga belum tersedia secara resmi di Indonesia. Sesuai dengan nama versinya dari sistem operasi Windows ini, versi ini ditujukan untuk menjadi fasilitas pendidikan penunjang bagi sekolah maupun universitas. Harga lisensi pasarannya sekitar 2 jutaan.

windows XP :

Nama "XP" adalah kependekan dari "Experience". windows Vista :

Windows Vista adalah sistem operasi berbasis grafis dari Microsoft yang digunakan pada komputer pribadi (PC), baik untuk pengguna rumahan maupun bisnis, pada komputer laptop, maupun media center.  Sebelum diumumkan dengan nama Windows Vista pada 22 Juli 2005, sistem operasi ini lebih dikenal dengan codename Longhorn

windows 8 :

windows 10 :

Windows 10 merupakan sistem operasi komputer pribadi yang dikembangkan oleh Microsoft sebagai bagian dari keluarga sistem operasi Windows NT. windows 11 :

Windows 11 diumumkan di acara Microsoft pada 24 Juni 2021, pukul 11 pagi ET (3 sore UTC).
