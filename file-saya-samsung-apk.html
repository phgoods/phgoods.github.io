5. cara ngatasi format video yg tidak support di tv samsung t4003? file video nya harus format apa? ​


Jika TV Samsung T4003 tidak dapat memutar file video tertentu karena format yang tidak didukung, Anda dapat mengubah format file video tersebut menjadi format yang didukung oleh TV tersebut. Format video yang didukung oleh TV Samsung T4003 antara lain:

AVI (DivX, XviD)

MP4 (MPEG4, H.264)

MKV (H.264, MPEG4, VP8)

WMV (VC1)

Untuk mengubah format file video, Anda dapat menggunakan perangkat lunak konversi video, seperti HandBrake, Freemake Video Converter, atau Any Video Converter. Berikut ini adalah langkah-langkah umum untuk mengkonversi format file video menggunakan HandBrake:

Unduh dan instal HandBrake dari situs web resminya.Buka HandBrake dan pilih file video yang ingin Anda konversi.Pilih format keluaran yang diinginkan, seperti MP4 atau MKV, pada menuKonfigurasi pengaturan keluaran, seperti resolusi, bit rate, dan codec, jika diperlukan.Klik tombol "Start" untuk memulai konversi. Proses konversi dapat memakan waktu tergantung pada ukuran file video yang ingin Anda konversi dan kecepatan komputer Anda.Setelah konversi selesai, salin file video yang baru dibuat ke perangkat penyimpanan eksternal yang dapat terhubung ke TV Samsung T4003, seperti flash drive atau hard drive eksternal.Sambungkan perangkat penyimpanan eksternal ke TV Samsung T4003 dan pilih file video yang ingin Anda putar.

Dengan mengubah format file video menjadi format yang didukung oleh TV Samsung T4003, Anda seharusnya dapat memutar video tanpa masalah. Pastikan juga untuk memperhatikan resolusi video dan bit rate yang Anda pilih saat mengkonversi, karena kualitas video dapat berdampak pada pengalaman menonton.

6. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.

PEMBAHASAN

DEGREE OF COMPARISON

Degree of comparison adalah perbandingan antara satu noun dengan noun lain berdasarkan perubahan pada bagian adjective. Terdapat 3 jenis dalam degree of comparison, yaitu;

POSITIVE DEGREE   (Kedua noun sebanding)

Rumus :  (noun + as + adj/adv as + noun)

COMPARATIVE DEGREE (Membandingkan 2 noun)

Rumus : [Adjective + er + than] Rumus saat berakhiran y : [Adjective + ier + than] Rumus saat syllable lebih dari 2 : [More + Adjective + than] Irregular form : [contoh : bad --> worse + than]

SUPERLATIVE DEGREE (Membandingkan 1 noun terhadap banyak noun)

Rumus : [Adjective + est] Rumus saat berakhiran y : [Adjective + iest] Rumus saat syllable lebih dari 2 : [Most + Adjective] Irregular form : [contoh : bad --> worst]PENJELASAN
