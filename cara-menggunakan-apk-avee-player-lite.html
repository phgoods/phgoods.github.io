2.Tekan Tombol On/Off
3.Tunggu Sampai Menyala
4.Selamat Menonton1.Pertama-tama persiapkan alat & bahan terlebih dahulu. Sepeti pesawat CD,dan  cassette.

2.Setelah siap tancapkan kabel steker CD pd sumber (PLN). 
3.sambungkan kabel RCA  dari CD ke TV.4.Kemudian tekan tombol ON/OFF pd CD untuk menghidupkan CD.
5.Setelah aktif tekan tombol “OPEN/CLOSE” untuk membuka tutup disk CD.
6.Setelah terbuka masukkan CD ke dalam tempat yg sudah tersedia, pastikan posisi tidak     terbalik.
7.Setelah selesai tutup kembali tutup disk CD dengan menekan tombol “OPEN/CLOSE”.
8.Dan tekan tombol “PLAY” untuk memutar film atau musik pada CD tersebut
9.Jika ingin berhenti tekan tombol “STOP” 
10.Kemudian tekan tombol “OPEN/CLOSE” untuk mengambil CD dr tempatnya.
11.Kemudian tutup kembali disk CD tersebut.
