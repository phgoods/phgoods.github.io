14. Apakah yang akan terjadi kalau pengguna sering install dan uninstall software


Tidak akan terjadi masalah apapun jika user terus-menerus reinstall suatu software, tapi ada kemungkinan akan ada error. Namun, kemungkinan itu sangat kecil. Biasanya software akan membuat file cache.

File cache ini digunakan untuk menyimpan data yang nantinya akan digunakan untuk memulai kembali aplikasi. Jadi, aplikasi tidak perlu memuat data yang banyak karena sudah tersimpan pada cache.

Bisa saja terjadi error saat memulai kembali aplikasi yang terus-menerus di-reinstall. Misalnya, file cache tersebut menumpuk dan aplikasi terkena freeze atau force close. Hal yang dapat mencegahnya adalah selalu membersihkan file cache sebelum meng-uninstall suatu software.

langkah-langkah untuk menginstal di komputer:

1. Buka Windows Explorer / My Computer, lalu temukan lokasi file installer (apakah di CD, di harddisk, dan sebagainya). Khusus untuk file installer CD, biasanya program instalasi akan langsung berjalan begitu CD dimasukkan ke drive. Jika tidak, maka Anda temukan lokasinya dengan Windows Explorer.

2. Klik dua kali file installer (biasanya berkestensi .exe dan dengan gambar ikon tertentu).

3. Tahapan instalasi biasanya cukup Anda selesaikan dengan menekan tombol Next dan Next hingga selesai. Biasanya di tahapan tersebut Anda diminta menyetujui EULA atau User Agreement, menentukan lokasi instalasi, dan menentukan opsi pembuatan ikon-ikon shortcutnya di desktop dan start menu.

4. Jika tahapan selesai, program biasanya akan dijalankan secara otomatis untuk pertama kalinya. Jika tidak, Anda dapat memeriksa dan menjalankan secara manual melalui menu Start > All Programs > nama program (program yang baru saja diinstal biasanya diberi highlight atau warna latar kuning).

langkah-langkah untuk menguninstal di komputer:

1. Klik menu Start

2. All Programs

3. Nama program

4. Uninstall

langkah-langkah untuk menginstal di handphone:

1. Buka appstore atau playstore

2. Ketik atau cari aplikasi yang inun di download

3. Tekan instal

langkah-langkah untuk menguninstal di handphone:

2. Geser ke tulisan "delete" atau logo kotak sampah. Bisa juga dengan masuk ke appstore atau playstore lalu cari aplikasi yang ingin dihapus dan tekan "copot pemasangan"
