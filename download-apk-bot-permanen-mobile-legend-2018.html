12. peluang seseorang untuk menjadi juara perlombaan mobile legend (ml) adalah 0,61. peluang seseorang tersebut tidak menjadi juara perlombaan mobile legend (ml) adalah..


Peluang seseorang untuk menjadi juara perlombaan mobile legend (ml) adalah 0,61. Peluang seseorang untuk tidak menjadi juara perlombaan mobile legend (ml) adalah 0,39.

Penjelasan dengan langkah-langkah:

Diketahui:

P(A) = Peluang seseorang untuk menjadi juara perlombaan mobile legend (ml)

P(A)' = Peluang seseorang untuk tidak menjadi juara perlombaan mobile legend (ml)

P (A) = 0,61

Ditanya:

P(A)' = ... ?

Jawab:

Jumlah keseluruhan Peluang = 100% = 1

P(A) = 0,61

P(A)' = 1 - 0,61

P(A)' = 0,39

Jadi, Peluang seseorang untuk tidak menjadi juara perlombaan mobile legend (ml) adalah 0,39.
