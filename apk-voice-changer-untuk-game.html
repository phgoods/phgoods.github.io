18. Many people had been trying that game jadikan passive voice


Bentuk passive dari "Many people had been trying that game" adalah...

Dilihat dari struktur kalimat nya, kalimat di atas termasuk ke dalam PAST PERFECT CONTINUOUS, dimana rumus nya :
Subject + had + been + V-ing + object.
=> Many people (had been trying) that game.

Maka rumus passive nya adalah..
Subject + had + been + being + V3 + by...

Catatan !!!
* subject di kalimat aktif berubah jadi object di kalimat pasif, begitu pula sebaliknya.

Maka jawabannya adalah..
=> That game HAD BEEN BEING TRIED by many people.
===============================
Mapel : Bahasa Inggris
Materi : Passive Voice
Kelas : 9
Kode Soal : 9.5
Kode Kategorisasi : 9.5.4
Kata Kunci : Past Perfect ContinuousJawaban:
That game had been being tried by many people

Berdasarkan soal, kalimat tersebut adalah kalimat Past Perfect Continues yg memiliki pola;
S + had + been + V+ing + O
active: Many people had been trying that game

Maka perubahan ke Passive;
S + had been + being + V3 + by (agent)
passive:
That game had been being tried by many people

Arti dari soal di atas adalah

Aktif: Dia bermain gim setiap malam.

Pasif: Gim (the game) dimainkan (is played) olehnya (by him) setiap malam (every night).

Kalimat di atas menggunakan simple present tense.

Rumus passive simple present tense adalah

(+) Object + am/is/are + verb 3 + by subject.

(-) Object + am/is/are + not + verb 3 + by subject.

Simple present tense digunakan untuk menyatakan suatu kegiatan/pekerjaan yang dilakukan berkali-kali/sudah menjadi suatu kebiasaan dan juga dapat digunakan untuk menyatakan suatu fakta seperti: The sun rises in the east (Matahari terbit di timur)
