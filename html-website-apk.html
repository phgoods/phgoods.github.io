16. jelaskan, Mengapa disebutkan bahwa HTML hanya untuk isi sedangkan CSS hanya untuk tampilan pada sebuah website?​


Jawaban & Penjelasan:

Dalam dunia pemrograman web, kita mengistilahkan HTML sebagai organ badan suatu halaman website dan CSS sebagai kulit yang mempercantik serta JS sebagai rangka badan yang menggerakkan dan memberi fungsi interaktif kepada website tersebut. HTML sudah menjadi balik layar dari website yang kita kunjungi, coba saja view-source. HTML-lah yang digunakan untuk membuat tatanan sebuah halaman web. Oleh karena itu, HTML dikatakan 'hanya untuk isi' yang berarti menjadi dasar suatu halaman website. Sedangkan CSS dikatakan 'hanya untuk tampilan' karena CSS digunakan untuk mendesain halaman website yang dibangun dengan HTML. CSS sangat berguna dalam membuat tampilan halaman website menarik, mulai dari tampilan teks, gambar hingga latar belakang suatu halaman website.
