12. Apa Kegunaan Termux, Dan Apa Keuntungan Dan Kerugiannya??​


Termux bukanlah aplikasi media sosial. bukan juga aplikasi yang digunakan untuk chatting apalagi untuk gaming. Dalam Google Playstore, Anda akan menemukan penjelasan bahwa Termux adalah terminal emulator dan environment Linux berbasis Linux Debian.

Anda yang baru mendengar istilah ini mungkin bingung, apa itu terminal emulator? Secara mudahnya, terminal emulator itu semacam Command Prompt (CMD) pada Windows atau Linux. Jika CMD adanya di Windows atau Linux, terminal emulator ini adanya di sistem operasi Android. Terminal ini digunakan untuk memasukkan, menampilkan, dan mencetak data.

Kelebihan dari Termux adalah aplikasi ini bisa langsung dipasang di smartphone Android Anda tanpa perlu proses rooting atau pengaturan lainnya. Termux tidak membutuhkan itu. Aplikasi ini juga mendukung perangkat Android berbasis 32bit ataupun 64bit.

Aplikasi ini dibuat oleh Fredik Fornwall. Sejak dirilis pertama kali pada 30 Juni 2015 di Google PlayStore, Termux telah diunduh oleh lebih dari 10 juta pengguna android. Hingga tulisan ini diturunkan, Termux telah mencapai versi 0.101 dan akan terus mengalami pembaharuan.

30. Jelaskan Secara Detail pengertian termux dan penjelasanya!​


Termux adalah salah satu aplikasi android terminal emulator dan linux environtment yang dapat di gunakan tanpa root dan juga root.

Ada banyak kegunaan termux, tentu nya untuk mepermudah user dalam beraktivitas seperti menggunakan bot atau pesan otomatis, bermain game, hacking, seperti : hack situs/website atau biasa disebut juga dengan deface, hack medsos (media sosial), Facebook, Instagram, WhatsApp, hack cctv, hack Video Tron/Layar papan reklame jalan, mengendalikan Laptop/PC, hp, gadget, android/Smartphone orang lain baik jarak dekat maupun jarak jauh, membuat dan mengirimkan virus /Virtex yang dapat membuat hp, gadget, android/Smartphone sih penerima menjadi lambat-loading, lemot, lelet, dan WhatsAppnya keluar sendiri atau eror, membuat dan mengirimkan Spam mulai dari spam telfon, pesan : sms, chat, ngerjain teman dan lain sebagainya.

*Trik Termux* ⓉⓔⓂⓐⓝ

Untuk mematikan atau memberhentikan tools tekan tombol volume bawah (-) + C secara bersamaan.

Untuk memunculkan commands/perintah sebelumnya tekan tombol volume bawah + P.

Menampilkan Tombol *ESC, HOME, CTRL dan lainnya* Tekan Tombol volume atas + Q.

Ketik *clear* Untuk Membersihkan tampilan layar.

Ketik *exit* untuk keluar dari aplikasi Termux.

Ketik *cp* Untuk mengcopy suatu file.

Ketik *history* Untuk mengetahui perintah-perintah yang sudah digunakan sebelumnya.

*Keterangan :*

- *apt* dan *pkg* fungsinya sama, yang membedakan hanya tulisan, huruf dan bacaannya.

- *&&* : untuk menggabungkan 2 perintah command-Termux.

- *-y* : untuk melanjutkan penginstalan tools tanpa harus mengetik *y ( y/n ) yes or no*, dalam bahasa Indonesia : *ya atau tidak*.
