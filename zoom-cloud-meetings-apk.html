7. 11. Komponen dari dua data atau lebih infrastruktur cloud disebut... a. Hybrid cloud b. Community cloud c. On-site-cloud d. Private cloud e. Public cloud​


Jawaban:

A. Hybrid cloud

Penjelasan:

Hybrid cloud adalah jenis infrastruktur cloud yang terdiri dari dua atau lebih komponen dari jenis infrastruktur cloud yang berbeda, seperti public cloud, private cloud, atau on-premises cloud. Dalam hybrid cloud, setiap jenis infrastruktur cloud tetap terpisah, tetapi dihubungkan melalui jaringan yang aman dan dapat berkomunikasi dengan lancar satu sama lain. Dengan menggunakan hybrid cloud, organisasi dapat memanfaatkan keuntungan dari setiap jenis infrastruktur cloud yang tersedia, misalnya keamanan dan kontrol yang lebih baik dari private cloud dan skalabilitas dan elastisitas dari public cloud. Hybrid cloud juga memberikan fleksibilitas bagi organisasi untuk memilih infrastruktur cloud yang paling cocok dengan kebutuhan bisnis mereka serta memungkinkan organisasi untuk memindahkan data dan aplikasi dari satu infrastruktur cloud ke infrastruktur cloud lainnya jika diperlukan.

Jawaban:

Cloud storage adalah ruang penyimpanan di internet yang dapat digunakan oleh pengguna untuk menyimpan file-file miliknya. Teknologi ini memungkinkan kita sebagai pengguna menempatkan file-file yang ada di internet. Cara ini akan memungkinkan pengguna memiliki backup file dan dapat mengaksesnya jika diperlukan.

Cloud adalah gabungan pemanfaatan teknologi komputer ('komputasi') dan pengembangan berbasis Internet ('awan'). Cloud adalah metafora dari internet, sebagaimana awan yang sering digambarkan di diagram jaringan komputer. Sebagaimana awan dalam diagram jaringan komputer tersebut, Cloud dalam Cloud Computing juga merupakan abstraksi dari infrastruktur kompleks yang disembunyikannya.

Perbedaan:

• Cloud storage membutuhkan kapasitas penyimpanan yang lebih besar, sedangkan cloud computing membutuhkan keberadaan memori dan cpu yang lebih banyak untuk melakukan pemrosesan.

• Cloud computing lebih banyak digunakan pada kebutuhan tingkat lanjut, baik bisnis atau aplikasi tertentu. Sedangkan cloud storage penggunaan penyimpanannya lebih ke data pribadi, kelompok atau pekerjaan.

• Dalam hal pengelolaannya, cloud storage lebih sederhana untuk dilakukan. Sedangkan untuk cloud computing harus memiliki atribut tambahan dalam pengelolaannya seperti akses pengguna, common line interface dan lainnya.

19. Tuliskan pengertian zoom windows zoom center dan zoom extends


Jawaban:

Zoom adalah app/apk untuk live streming dengan banyak orng seperti vidio call

Jawaban:

Zoom dalam AutoCAD berfungsi untuk melihat obyek dalam area gambar di layar monitor dengan cara membesarkanya atau mengecilkanya,yang sering disebut dengan istilah zoom in dan zoom out,tetapi didalam autoCAD masih banyak lagi cara melihat sebuah obyek gambar tergantung metode zoom yang dipergunakan.

Metode-metode zoom dalam AutoCAD ini memiliki misi tampilan yang berbeda-beda antara metode satu dengan yang lainya,tetapi tetap pada satu tujuan yakni zoom in dan zoom out sebuah obyek gambar pada area drawing di layar monitor.

Mengapa melakukan zoom menggunakan metode tertentu.

Seperti saya utarakan diatas bahwa metode-metode yang ada pada perintah zoom itu memilki hasil tampilan yang berbeda beda,seperti contoh ketika kita akan menampilkan hanya salah satu obyek saja maka kita akan memerlukan metode yang berbeda dengan ketika kita akan menampilkan seluruh obyek dalam satu layar,untuk lebih jalas dan detailnya maka akan saya uraikan beserta tutorial cara menggunakan metode tersebut masing-masing.

Metode-metode zoom dalam AutoCAD.

Banyak jenis dan ragam cara melakukan zoom dalam AutoCAD,bisa menggunakan scroll pada mouse,zoom toolbar,command dan masih banyak yang lainya ( lihat postingan perintah zoom dalam AutoCAD ),tetapi yang akan kita coba untuk mengulasnya adalah zoom yang menggunakan zoom toolbar dan command yang memilki kesamaan baik cara maupun hasil yang ditampilkan.

All ( Z+A).

Metode zoom all ini kita akan menghasilkan seluruh obyek yang ada pada sebuah file drawing akan muncul dilayar komputer kita sesuai dengan titik koordinat dimana obyek tersebut pertama dan titik koordinat obyek lain yang pernah dibuat. ( tutorial cara menggunakan zoom metode All ).

Center. ( Z+C ).

Metode zoom center ini akan menghasilkan tampilan gambar berdasarkan titik pusat koordinat gambar yakni 0,0,maka metode center ini akan sangat berfungsi jika kita menentukan titik awal sebelum memulai gambar sebuah pekerjaan pada titik koordinat 0,0. ( tutorial cara menggunakan zoom metode center ).

Dynamic. ( Z+D ).

Metode zoom dynamic ini akan menghasilkan sebuah hasil pembesaran yang bisa kita ulang ulang sampai mendapatkan tampilan yang sesuai dengan keinginan kita. ( tutorial cara menggunakan zoom metode Dynamic ).

Extents ( Z+E ).

Metode zoom extend ini menghasilkan seluruh obyek akan tampil dan termuat dalam satu layar monitor,dengan menggunakan metode ini kita akan menghasilkan tampilan obyek lebih besar atau lebih kecil dari sebelumnya. ( tutorial cara menggunakan zoom metode Extend ).

Previous ( Z+P ).

Zoom metode previous ini memiliki tujuan untuk mengembalikan tampilan sebuah obyek seperti semula sebelum kita melakukan perintah zoom,jadi setelah kita melakukan perintah zoom dengan metode apapun dan setelah selesai ingin melihat tampilan semula sebelum di zoom tadi maka menggunakan metode previous inilah salahsatu solusinya. ( tutorial cara menggunakan zoom metode Previous ).

Scale ( Z+S ).

Melakukan zoom menggunakan metode scale ini berarti kita akan menghasilkan sebuah gambar obyek yang bisa jadi lebih besar atau bisa jadi lebih kecil,tergantung pada scale factor zoom yang kita masukan apakah lebih besar dari scale factor zoom tampilan sebelumnya atau sebaliknya. ( tutorial cara menggunakan zoom metode Scale ).

Window ( Z+W ).

Melakukan perintah zoom dengan menggunakan metode window ini kita akan menampilkan obyek tertentu dalam area gambar berdasarkan kotak area yang kita inginkan,dan kita bisa melakukanya berulang ulang sama area tersebut sesuai dengan yang kita inginkan. ( tutorial cara menggunakan zoom metode window ).

Object. ( Z+O ).

Zoom metode obyek ini bertujuan untuk memperbesar tampilan sebuah obyek tertentu yang telah kita pilih sebelumnya,jadi tampilan yang akan diperbesar hanya obyek yang kita pilih saja. ( tutorial cara menggunakan zoom metode Object ).

Real time ( Z+R ).

Metode real time pada saat melakukan zoom seperti halnya ketika kita melakukan zoom dengan menggunakan scroll pada mouse yakni zoom in dengan mendorong roda kedepan dan zoom out dengan mendorong roda scroll kebelakang. ( tutorial cara menggunakan zoom metode Real time ).

In.

Metode ini hanya berfungsi untuk melakukan pembesaran tampilan sebuah gambar.

Out.

Metode ini hanya berfungsi untuk melakukan pengecilan tampilan sebuah gambar.

Demikian metode-metode zoom yang dapat kita pergunakan untuk melihat tampilan gambar pada area kerja di layar monitor

21. 4.Apakah yang Anda ketahui dengan aplikasi berbasis cloud (cloud based application) ? 5. Apakah keuntungan dari penggunaan apkikasi berbasis cloud ?


Aplikasi berbasis cloud (cloud based application) adalah aplikasi yang disediakan oleh perusahaan penyedia layanan cloud. Maksud dari layanan cloud adalah aplikasi yang kita gunakan berada di internet dan tidak terinstall di komputer kita. Untuk mengakses aplikasi tersebut kita memerlukan akses internet.

Keuntungan dari penggunaan aplikasi berbasis cloud adalah keamanan data dan kita tidak dipusingkan untuk melakukan perawatan terhadap aplikasi tersebut karena sudah menjadi tanggung jawab penyedia layanan cloud tersebut.

Pembahasan

Setelah mengetahui apa itu aplikasi berbasis cloud dan keuntungan dari menggunakan aplikasi cloud, mari kita sebutkan jenis-jenis dari aplikasi berbasi cloud, beberapa jenis aplikasi cloud adalah sebagai berikut

Infrastructure as a Service contohnya adala Amazon EC2, Telkom Cloud.Platform as a Service contoh aplikasinya adalah Windows AzureSoftware as a Service aplikasi Office 365 merupakan contoh aplikasi cloud jenis Software as a Service

Penggunaan aplikasi cloud tentu sangat membantu seorang pengusaha, karena bisa lebih fokus untuk pengembangan usahanya dan tidak dipusingkan dengan mengurus komputer dan aplikasi-aplikasi pendukungnya. Selain contoh-contoh diatas, masih ada contoh aplikasi cloud yang sering kita gunakan yaitu email dan media sosial.
