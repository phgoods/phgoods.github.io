Adi dan Budi merupakan sahabat karib. Malam itu mereka berada dirumah masing-masing. Jarak kedua rumah adalah 2 Km. Adi mengirim pesan singkat kepada budi bahwa dia sedang beridiri menghadap rumah budi dan bermain pistol laser hijau yang kuat dan ditembakkan dengan sudur elevasi 75° ke awan yang berada dilangit antara kedua rumahnya sehingga mengenai awan. Budi beranjak berdiri di depan rumah sambil mengamati titik hijau di awan menggunakan klinometer dan terbaca sudut yang terbentuk 45° . Maka tinggi awan yang ditembak adi adalah (1/3) ( 3 + √3 ) km

PEMBAHASAN

Sebelumnya kita mesti mengetahui prinsip dasar dari trigonometri.

Jikalau di ketahui segitiga siku-siku maka bisa di buat :

sinus sebuah sudut = sisi di depan sudut / sisi miring

kosinus sebuah sudut = sisi di samping sudut / sisi miring

tangen sebuah sudut = sisi di depan sudut / sisi di samping sudut

Marilah kita gunakan prinsip dasar ini untuk menyelesaiakan soalnya.

Mula-mula kita gambarkan dulu persoalannya dalam gambar seperti di lampiran.

tan 75° = CW / CA

tan 75° = h / x

x = h / tan 75°


tan 45° = CW / BC

tan 45° = h / ( 2 - x )

1 = h / ( 2 - x )

h = 2 - x

h = 2 - h / tan 75°

h + h / tan 75° = 2

h ( 1 + 1 / tan 75° ) = 2

h = 2 / ( 1 + 1 / tan 75° )

h = ( 2 tan 75° ) / ( 1 + tan 75° )

Berikutnya kita mencari nilai tan 75° dengan cara :

tan 75° = tan ( 45° + 30° )

tan 75° = ( tan 45° + tan 30° ) / ( 1 - ( tan 45° ) ( tan 30° ) )

tan 75° = ( 1 + (1/3)√3 ) / ( 1 - (1/3)√3 )

tan 75° = 2 + √3

Nilai ini kita masukkan kembali ke persamaan di atas :

h = ( 2 tan 75° ) / ( 1 + tan 75° )

h = ( 2 (2 + √3 )) / ( 1 + (2 + √3) )

h = ( 4 + 2√3 ) / ( 3 + √3 )

h = (1/3) ( 3 + √3 ) km

24. Adi dan Budi merupakan sahabat karib. Malam itu mereka berada di rumah masing-masing. Jarak kedua rumah adalah 2 km. Adi mengirim pesan singkat kepada Budi bahwa dia sedang berdiri menghadap rumah Budi dan bermain pistol laser hijau yang kuat dan ditembakkan dengan sudut elevasi 75° ke awan yang berada di langit antara kedua rumahnya sehingga mengenai awan. Budi beranjak berdiri di depan rumah sambil mengamati titik hijau di awan menggunakan klinometer dan terbaca sudut yang terbentuk 45°. Maka tinggi awan yang ditembak Adi adalah ....​


Adi dan Budi merupakan sahabat karib. Malam itu mereka berada di rumah masing-masing.  Jarak kedua rumah adalah 2 km. Adi mengirim pesan singkat kepada Budi bahwa dia sedang  berdiri menghadap rumah Budi dan bermain pistol laser hijau yang kuat dan ditembakkan  dengan sudut elevasi 75° ke awan yang berada di langit antara kedua rumahnya sehingga mengenai awan. Budi beranjak berdiri di depan rumah sambil mengamati titik hijau di awan  menggunakan klinometer dan terbaca sudut yang terbentuk 45°. Maka tinggi awan yang  ditembak Adi adalah [tex]\bold{\frac{3+\sqrt{3}}{6}}[/tex]

Untuk mencari tinggi awan maka kita menggunakan perbandingan Trigonometri segitiga siku-siku

Rumus Trigonometri :

Sin A = [tex]\frac{depan}{miring}[/tex]

Cos A = [tex]\frac{samping}{miring}[/tex]

Tan A = [tex]\frac{depan}{samping}[/tex]

Pembahasan :

Diketahui :

Jarak kedua rumah 2 km

Sudut elevasi Adi = 75°

Sudut elevasi Budi = 45°

Ditanya :

Tinggi Awan ?

Dijawab :

Nilai Tan 75° = 2 + √3

Nilai Tan 45° = 1

Kita perhatikan gambar terlampir

Jarak Adi dan budi (BC) = 2 km

Jarak Budi ke bawah awan (BD) = x km

Jarak Adi ke bawah awan (CD) = (2 - x) km

Untuk menghitung ini kita gunakan Rumus perbandingan Tangen

Tan A = [tex]\frac{depan}{samping}[/tex]

Tan 45 = [tex]\frac{AD}{BD}[/tex]

1 = [tex]\frac{AD}{x}[/tex]

AD = x

Tan A = [tex]\frac{depan}{samping}[/tex]

Tan 75 = [tex]\frac{AD}{CD}[/tex]

2 + √3 = [tex]\frac{AD}{2 - x}[/tex]

(2 + √3)(2 - x) = AD

AD = 4 - 2x + 2√3 - x√3

Karena Tinggi awan sama maka kita bisa hitung nilai x

AD = AD

x = 4 - 2x + 2√3 - x√3

x + 2x + x√3 = 4 + 2√3

3x + x√3 = 4 + 2√3

x(3 + √3) = 4 + 2√3

x = [tex]\frac{4+2\sqrt{3}}{3+\sqrt{3}}[/tex]

x = [tex]\frac{4+2\sqrt{3}}{3+\sqrt{3}}[/tex] x [tex]\frac{3-\sqrt{3}}{3-\sqrt{3}}[/tex]

x = [tex]\frac{(4+2\sqrt{3})(3-\sqrt{3})}{9+3}[/tex]

x = [tex]\frac{12-4\sqrt{3}+6\sqrt{3}-6}{12}[/tex]

x = [tex]\frac{6+2\sqrt{3}}{12}[/tex]

x = [tex]\frac{3+\sqrt{3}}{6}[/tex]

Selanjutnya kita cari tinggi awan

AD = x

AD = [tex]\frac{3+\sqrt{3}}{6}[/tex]

∴ Jadi tinggi awan yang ditembak Adi adalah [tex]\frac{3+\sqrt{3}}{6}[/tex]
