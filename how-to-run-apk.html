4. melvin can run a speed of 150 metres per minute. How long does it take for him to run a distance of 900 metres?


Penjelasan dengan langkah-langkah:

terjemahan soal:

melvin bisa berlari dengan kecepatan 150 meter per menit. Berapa lama baginya untuk berlari sejauh 900 meter?

Diketahui:

kecepatan ( k )

= 150 m/menit

jarak ( j )

= 900 m

Ditanya:

waktu ( t )?

langkah pengerjaan:

waktu ( t )

= J ÷ k

= 900 m ÷ 150 m

= 6 menit

Jawaban & kesimpulan:

jadi, waktu yang diperlukan Melvin adalah 6 menit
