Penjelasan dengan langkah-langkah:

1. 300 ekor bebek

96 ekor mati

Dan telur menetas 67

300 - 96

= 204

= 204 + 67

= 271

Jumlah bebek bu Nif adalah 271 ekor

___________________________________

2. (25 × (-7)) - ((-7) × 8)

= ( -175 ) - ( -56 )

= ( - 175 ) + 56 ( karena negatif bertemu negatif jadi positif )

= ( -119 )

___________________________________

Jika ada kesalahan koreksi ya

1) Bu Nif memiliki 300 ekor bebek. Karena ada virus, bebek mati 96 ekor dan telur menetas 67. Berapa jumlah bebek Bu Nif sekarang?

Penyelesaian :

Logika = 300 bebek kurangi yang mati 96 tapi di tambah yang telur menetas sebanyak 67.

Maka...

=> 300 - 96 + 67

=> 204 + 67

=> 271 ekor

2) Selesaikan OBB berikut :

a. (25 x (-7)) - ((-7) x 8)

(25 × (-7)) - ((-7) × 8)

=> (25 × (-7)) - ((-7) × 8)

=> ( -175 ) - ( -56 )

=> ( - 175 ) + 56

=> -119

Ingat !

=> negatif bertemu negatif jadi positif

=> positif bertemu negatif hasilnya negatif
