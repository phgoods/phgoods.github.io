5. pertanyaan dan jawaban untuk pengecilan ukuran


Apakah pengertian atau definisi dari pengecilan ukuran?

Pengecilan ukuran adalah proses atau operasi penghancuran dan pemotongan bahan tertentu dengan tujuan mengurangi ukuran bahan dengan kerja mekanis, yakni memecahnya menjadi partikel-partikel lebih kecil. Dalam hal pengurangan ukuran, upaya telah dilakukan untuk menggunakan perangkat mekanis tanpa mengubah struktur kimia bahan, serta keseragaman ukuran dan bentuk unit partikel diperlukan pada akhir proses. Bahan baku terkadang lebih besar dari yang kita butuhkan. Nah sebab itu, ukuran bahan ini harus dikurangi atau dikecilkan.

Pembahasan

Pada umumnya, pengecilan ukuran sering digunakan untuk menunjukkan manipulasi, pembagian, atau kerusakan mekanis suatu material menjadi potongan-potongan (partikel) yang lebih kecil tanpa harus mengubah sifat kimianya. Pengecilan ukuran dilakukan guna memperbesar permukaan padatan, sehingga pencampuran dapat terjadi secara merata ketika bahan lain ditambahkan. Proses pengecilan ukuran dibagi menjadi dua, hal tersebut tergantung pada bahan tersebut, apakah berupa bahan cair atau padat. Jika bahan padat, maka proses atau operasi pengecilan disebut dengan penghancuran dan pemotongan. Jika bahan cair, maka disebut dengan emulsifikasi (atomisasi).

Terdapat cara atau metode yang digunakan untuk melaksanakan pengecilan ukuran antara lain pemotongan atau perajangan, kompresi (pemukulan/penggerusan/penumbukan), dan juga menggiling (shearing). Berikut ini adalah beberapa tujuan pengecilan ukuran, antara lain:

Memberikan kemudahan untuk mengekstraksi unsur tertentu dan struktur komposisi.Menyesuaikan dengan kebutuhan/keperluan spesifikasi suatu produk atau memperoleh bentuk tertentu sesuai dengan yang diinginkan.Menambah dan meningkatkan luas permukaan padatan.Memberikan kemudahan dalam pencampuran bahan secara merata.

11. bagaimana cara menyimpan file atau format gif pada program atau apk adobe photoshop​


Cara menyimpan file atau format gif pada program atau aplikasi adobe photosop adalah yaitu dengan menggunakan perintah save as dengan cara klik menu file yang ada pada menu bar atas, kemudian pilih Save As.

Sesaat setelah kamu menekan tombol save as, akan muncul sebuah jendela save as yang berisi nama file dan tipe file yang akan kamu simpan.

Pembahasan

Nah pada pilihan tipe file atau dalam aplikasi akan muncul tulisan format, kamu pilih format CompuServe yaitu dengan extensi .gif.  

Pilih OK, maka gambar kamu akan tersimpan dengan format GIF, dan kamu bisa mengecek dan membukanya di folder tempat kamu menyimpan foto tersebut

File Gif biasanya digunakan untuk menyimpan gambar yang bersifat animasi atau gambar yang bergerak untuk ditampilkan di presentasi atau suatu website.
