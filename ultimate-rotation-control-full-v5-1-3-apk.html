Untuk mengubah izin baca menjadi kontrol penuh menggunakan command prompt, pertama-tama Anda perlu membuka command prompt dengan cara klik start, ketik cmd pada kolom pencarian, lalu tekan Enter. Setelah command prompt terbuka, Anda dapat menggunakan perintah `cacls` atau `icacls` untuk mengubah izin akses pada suatu file atau folder.

Contoh:

cacls nama_file /e /p users:f

Atau:

icacls nama_file /grant users:(OI)(CI)F

Perintah di atas akan memberikan izin kontrol penuh kepada pengguna pada file bernama `nama_file`. Jika Anda ingin memberikan izin kontrol penuh kepada pengguna lain, ganti `users` dengan nama pengguna yang dimaksud.

Perlu diingat bahwa hanya pengguna dengan hak akses yang tepat yang dapat mengubah izin akses pada file atau folder. Jika Anda mengalami masalah saat mencoba mengubah izin akses, cobalah untuk login sebagai administrator dan coba kembali.
